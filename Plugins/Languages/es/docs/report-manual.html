Manual del Diseñador de Informes

Estructura del informe

Un informe consta de tres secciones:

  • Encabezado – se imprime en la parte superior de cada página
  • Cuerpo – contiene las tablas de datos
  • Pie de página – se imprime en la parte inferior de cada página

Cada sección puede editarse a través de la pestaña Diseño en el editor de informes.

Sección de encabezado

El encabezado tiene tres zonas: Izquierda, Centro y Derecha.

Cada zona puede contener:

  • Campos – campos de base de datos que se rellenan automáticamente
  • Texto libre – texto introducido manualmente
  • Imágenes – p. ej., un logotipo de empresa (cargar a través del administrador de imágenes)

Las filas se pueden añadir o eliminar mediante la barra de herramientas.

Tablas del cuerpo

El cuerpo contiene una o más tablas.

AcciónDescripción
Añadir columnaMediante el símbolo + o arrastrando desde la lista de campos
Ancho de columnaAjustar arrastrando el borde de la columna
Orden de columnasCambiar arrastrando y soltando
Eliminar columnaEn el editor de columnas mediante el símbolo de eliminación

Cada tabla está vinculada a un conjunto de datos (p. ej., horario de trabajo, direcciones).

Combinar campos (Merged Fields)

Los campos combinados permiten múltiples vinculaciones de datos en una sola columna.

  • En el editor de columnas, hacer clic en Añadir campo para agregar más campos a la misma columna
  • Los valores se muestran uno debajo del otro o con un separador
  • El separador se puede configurar en el diálogo de campo (p. ej., coma, espacio, salto de línea)

Pie de tabla

Debajo de cada tabla se puede mostrar un pie de página.

  • Campos SUM: las columnas numéricas se pueden sumar automáticamente
  • Etiqueta: la descripción de la suma puede mostrarse u ocultarse
  • Se activa en el editor de columnas bajo la pestaña Pie de página

Filas de texto libre

Se pueden insertar filas de texto libre antes o después de las tablas de datos.

PropiedadDescripción
PosiciónSeleccionable Antes de los datos o Después de los datos
ContenidoTexto libre, p. ej., notas, condiciones o campos de firma
EstiloFuente, tamaño y alineación configurables

Formato

Cada celda y cada texto libre se puede formatear individualmente:

OpciónDescripción
FuenteFuentes seleccionables para la celda
Tamaño de fuenteTamaño en puntos (pt)
Negrita / Cursiva / SubrayadoMediante los botones B / I / U
Color de textoColor de libre elección
AlineaciónIzquierda, centrado o derecha

Bordes de celda

Los bordes de cada celda se pueden configurar individualmente.

PropiedadOpciones
LadosSuperior, derecho, inferior, izquierdo (individualmente o todos)
Estilo de líneaFino, medio, grueso, discontinuo, doble
Color de líneaDe libre elección

Los bordes se configuran en el editor de columnas bajo Bordes.

Configuración de página

En la pestaña General se pueden ajustar las opciones de impresión:

  • Orientación: Vertical (Portrait) u horizontal (Landscape)
  • Tamaño de papel: A4, A3, B4 (JIS), B5 (JIS) o Carta
  • Márgenes de página: Superior, derecho, inferior, izquierdo en mm

Opciones de presentación (Diseñador)

En la pestaña Diseño, cuando la fuente de datos es Horario de trabajo (Schedule), hay opciones adicionales disponibles:

  • Consolidar entradas por día: Varias entradas del mismo día se agrupan en una sola fila
  • Mostrar período completo: Se muestran todos los días del período, incluso si no hay entradas

Fuentes de datos

Fuentes de datos disponibles para informes:

Fuente de datosDescripción
Horario de trabajo (Schedule)Turnos y horarios de trabajo
Direcciones (Lista)Todas las direcciones como tabla
Dirección (Detalle)Dirección individual
AusenciasResumen de ausencias

La fuente de datos se selecciona en la pestaña «Fuente de datos» y determina los campos disponibles.
