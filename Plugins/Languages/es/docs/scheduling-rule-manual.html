Reglas de planificación - Explicaciones

Las reglas de planificación definen las condiciones marco para la programación automática de turnos. Cada regla establece cómo se distribuyen los tiempos de trabajo, los períodos de descanso y las capacidades para un grupo o configuración determinados.

Aplicación

Las reglas de planificación se asignan a un contrato. De este modo, los límites definidos se aplican a todos los empleados con dicho contrato durante la programación automática de turnos.

Los valores predeterminados aquí establecidos provienen de las Reglas de planificación (valores predeterminados) en la configuración. Al crear una nueva regla, estos valores se adoptan automáticamente y pueden ajustarse de forma individual.
