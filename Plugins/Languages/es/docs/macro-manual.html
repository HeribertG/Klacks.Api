Manual de Scripting de Macros

Introducción

El editor de macros permite escribir scripts que se utilizan como base de cálculo para servicios, actividades laborales, etc.

Ejemplos de uso:

  • Cálculo exacto de horas para vacaciones o servicio militar según el grado de ocupación
  • Cálculo de recargos por trabajo nocturno, fines de semana y días festivos
  • Adaptación flexible a nuevos requisitos

Nota: El intérprete de scripts se ejecuta en un entorno sandbox seguro sin acceso al sistema de archivos ni a la red. No es posible ejecutar código malicioso.

Estructuras de control

Importante: Las palabras clave como ENDIF, ENDFUNCTION, ENDSUB deben escribirse juntas (no END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "grande"
ELSEIF x > 5 THEN
    OUTPUT 1, "mediano"
ELSE
    OUTPUT 1, "pequeño"
ENDIF

IF en una sola línea

IF x > 10 THEN OUTPUT 1, "grande" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Fin de semana"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Día laborable"
    CASE ELSE
        OUTPUT 1, "Desconocido"
END SELECT

Bucle FOR-NEXT

FOR i = 1 TO 10
    suma += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

Bucle DO-LOOP

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Funciones y procedimientos

Importante: Las funciones (FUNCTION) y los procedimientos (SUB) deben definirse antes de la primera llamada. El intérprete lee el código de arriba a abajo: una función o procedimiento solo es conocido después de que su definición haya sido procesada.

SUB (procedimiento sin valor de retorno)

SUB calcular(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (función con valor de retorno)

FUNCTION duplicar(x)
    duplicar = x * 2
ENDFUNCTION

DIM resultado
resultado = duplicar(21)
OUTPUT 1, resultado

Ejemplo: Cálculo de recargos

Calcula recargos por trabajo nocturno, días festivos y fines de semana con tratamiento correcto de turnos que cruzan la medianoche (división en segmentos).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Calcular horas nocturnas (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Sá = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Do = 7 (ISO-8601)

    ' Tasa más alta para horas nocturnas
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Tasa más alta para horas diurnas
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Calcular el día de la semana del día siguiente (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Turno que cruza la medianoche - 2 segmentos
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Turno diurno normal
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Explicación

  • División en segmentos: Para turnos que cruzan la medianoche (p. ej., 22:00-06:00), el turno se divide en dos segmentos
  • CalcSegment: Calcula el recargo para un segmento teniendo en cuenta los recargos nocturnos, por festivos y por fin de semana
  • Tasas separadas para sábado/domingo: sarate y sorate permiten recargos diferentes para sábado y domingo
  • Recargo más alto: Siempre se aplica el recargo más alto aplicable
  • holidaynextday: Tiene en cuenta si el día siguiente es festivo (importante para turnos nocturnos)
