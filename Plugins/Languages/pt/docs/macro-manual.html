Manual de scripting de macros

Introdução

Com o editor de macros pode criar scripts que servem de base de cálculo para trabalhos, tipos de emprego, entre outros.

Exemplos de uso:

  • Cálculo preciso de férias e serviço militar com base na taxa de emprego
  • Cálculo de suplementos para trabalho noturno, fins de semana e feriados
  • Adaptação flexível a novos requisitos

Nota: O interpretador de scripts funciona num ambiente sandbox seguro sem acesso ao sistema de ficheiros ou à rede. Não é possível executar código malicioso.

Estruturas de controlo

Importante: Palavras-chave como ENDIF, ENDFUNCTION, ENDSUB devem ser escritas sem espaço (não usar END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "grande"
ELSEIF x > 5 THEN
    OUTPUT 1, "médio"
ELSE
    OUTPUT 1, "pequeno"
ENDIF

IF numa linha

IF x > 10 THEN OUTPUT 1, "grande" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "fim de semana"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "dia útil"
    CASE ELSE
        OUTPUT 1, "desconhecido"
END SELECT

Ciclo FOR-NEXT

FOR i = 1 TO 10
    summe += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

Ciclo DO-LOOP

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Funções e procedimentos

Importante: Funções (FUNCTION) e procedimentos (SUB) devem ser definidos antes da primeira chamada. O interpretador lê o código de cima para baixo.

SUB (procedimento sem valor de retorno)

SUB calcular(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (função com valor de retorno)

FUNCTION duplicar(x)
    duplicar = x * 2
ENDFUNCTION

DIM resultado
resultado = duplicar(21)
OUTPUT 1, resultado

Exemplo: Cálculo de suplementos

Calcula suplementos noturnos, de feriados e de fins de semana com tratamento correto de turnos que atravessam a meia-noite (divisão em segmentos).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Calcular horas noturnas (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6
    IsSunday = WeekdayNum = 7

    ' Taxa máxima para horas noturnas
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Taxa máxima para horas diurnas
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Turno que atravessa a meia-noite - 2 segmentos
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Turno diurno normal
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Explicação

  • Divisão em segmentos: Turnos que atravessam a meia-noite (ex.: 22:00-06:00) são divididos em dois segmentos
  • CalcSegment: Calcula o suplemento do segmento considerando noite, feriados e fins de semana
  • Taxas de sábado/domingo: sarate e sorate permitem taxas diferentes para sábado e domingo
  • Taxa máxima: É sempre aplicada a taxa mais elevada
  • holidaynextday: Considera se o dia seguinte é feriado (importante para turnos noturnos)
