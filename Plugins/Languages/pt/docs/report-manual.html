Manual do designer de relatórios

Estrutura do relatório

O relatório é composto por três secções:

  • Cabeçalho – Impresso no topo de cada página
  • Corpo – Contém as tabelas de dados
  • Rodapé – Impresso no final de cada página

Cada secção pode ser editada no separador Designer do editor de relatórios.

Secção de cabeçalho

O cabeçalho tem três zonas: esquerda, centro, direita.

Cada zona pode conter:

  • Campos – Campos de base de dados preenchidos automaticamente
  • Texto livre – Texto introduzido manualmente
  • Imagens – Ex.: logótipo da empresa (carregado pelo gestor de imagens)

As linhas podem ser adicionadas ou removidas através da barra de ferramentas.

Tabela do corpo

O corpo contém uma ou mais tabelas.

OperaçãoDescrição
Adicionar colunaAtravés do ícone + ou arrastando da lista de campos
Largura da colunaArrastar a borda da coluna para ajustar
Ordem das colunasMover por arrastar e soltar
Remover colunaAtravés do ícone de eliminar no editor de colunas

Cada tabela está associada a um conjunto de dados (ex.: plano de trabalho, endereços).

Campos combinados (campos de junção)

Os campos de junção permitem definir múltiplas ligações de dados numa única coluna.

  • No editor de colunas, clique em Adicionar campo para adicionar mais campos à mesma coluna
  • Os valores são apresentados empilhados ou com separadores
  • O separador pode ser configurado no diálogo de campos (ex.: vírgula, espaço, nova linha)

Rodapé da tabela

Pode ser apresentado um rodapé abaixo de cada tabela.

  • Campos SOMA: Colunas numéricas podem ser somadas automaticamente
  • Rótulo: O rótulo da soma pode ser mostrado ou ocultado
  • Ativa-se no separador Rodapé do editor de colunas

Linhas de texto livre

Linhas de texto livre podem ser inseridas antes ou depois das tabelas de dados.

PropriedadeDescrição
PosiçãoEscolher Antes dos dados ou Depois dos dados
ConteúdoQualquer texto, ex.: notas, condições, espaço para assinatura
EstiloFonte, tamanho e alinhamento configuráveis

Formatação

Cada célula e texto livre pode ser formatado individualmente:

OpçãoDescrição
FonteFonte selecionável para a célula
Tamanho da fonteTamanho em pontos (pt)
Negrito / Itálico / SublinhadoAlternar com os botões B / I / U
Cor do textoCor livremente selecionável
AlinhamentoEsquerda, centro ou direita

Bordas das células

As bordas de cada célula podem ser configuradas individualmente.

PropriedadeOpções
LadoSuperior, direita, inferior, esquerda (individual ou todas)
Estilo da linhaFina, média, grossa, tracejada, dupla
Cor da linhaLivremente selecionável

As bordas são configuradas no separador Bordas do editor de colunas.

Configuração da página

No separador Geral pode ajustar as configurações de impressão:

  • Orientação: Retrato ou paisagem
  • Tamanho do papel: A4, A3, B4 (JIS), B5 (JIS) ou Letter
  • Margens da página: Superior, direita, inferior, esquerda (em mm)

Opções de visualização (Designer)

Para a fonte de dados Plano de trabalho (Schedule), o separador Designer mostra opções adicionais:

  • Agrupar entradas do mesmo dia: Mostra múltiplas entradas do mesmo dia numa única linha
  • Mostrar período completo: Mostra todos os dias do período, incluindo dias sem entradas

Fontes de dados

Fontes de dados disponíveis para relatórios:

Fonte de dadosDescrição
Plano de trabalho (Schedule)Turnos e horas de trabalho
Endereços (Lista)Todos os endereços em formato de tabela
Endereço (Detalhe)Endereço individual
AusênciasLista de ausências
Turnos (Shifts)Tabela de turnos
GruposLista de grupos
Modelo de contentorModelo para relatórios de contentores

A fonte de dados é selecionada no separador "Fonte de dados" e determina os campos disponíveis.
