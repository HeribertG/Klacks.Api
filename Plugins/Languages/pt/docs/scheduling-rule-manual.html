Regras de planeamento - Explicações

As regras de planeamento definem o enquadramento para o agendamento automático de turnos. Cada regra determina como as horas de trabalho, pausas e capacidades são distribuídas para um grupo ou configuração específica.

Aplicação

As regras de planeamento são atribuídas a contratos. Isto garante que os limites definidos são aplicados durante o agendamento automático de turnos a todos os colaboradores com esse contrato.

Os valores predefinidos aqui configurados são obtidos a partir das Regras de planeamento (valores predefinidos) nas configurações. Ao criar uma nova regra, estes valores são automaticamente herdados e podem ser ajustados individualmente.
