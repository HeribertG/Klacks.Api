תאריך קבוע עם הזזת יום בשבוע

פורמט: MM/DD+XX+WW או MM/DD+XX-WW

  • +XX = סימן + היסט (ימים) (00 עד 99)
  • + = יום השבוע הבא אחרי התאריך
  • - = יום השבוע הקודם לפני התאריך
  • WW = יום בשבוע (SU, MO, TU, WE, TH, FR, SA)
דוגמהמשמעות
11/01+00+THיום חמישי הראשון בנובמבר
11/22+00+THיום חמישי הרביעי בנובמבר (חג ההודיה)
05/25+00-MOיום שני האחרון במאי (יום הזיכרון)

פורמט כלל משנה

כללי משנה מאפשרים העברה אוטומטית של חג שחל ביום מסוים בשבוע.

פורמט: WW+X או WW-X, כללים מרובים מופרדים ב-;

  • WW = יום בשבוע (SU, MO, TU, WE, TH, FR, SA)
  • +X = הזז X ימים קדימה
  • -X = הזז X ימים אחורה
דוגמהמשמעות
SA+2;SU+1אם חל בשבת: +2 ימים (שני), אם חל בראשון: +1 יום (שני)
SU+1אם חל בראשון: מועבר לשני

הערה: כללי משנה משמשים בדרך כלל להעברת חגים שחלים בסוף שבוע ליום העבודה הבא.
