מדריך סקריפטים של מאקרו

מבוא

עורך המאקרו מאפשר לכתוב סקריפטים המשמשים כבסיס חישוב עבור שירותים, פעילויות עבודה וכו'.

דוגמאות שימוש:

  • חישוב מדויק של שעות לחופשה או שירות צבאי לפי דרגת התעסוקה
  • חישוב תוספות לעבודת לילה, סופי שבוע וימי חג
  • התאמה גמישה לדרישות חדשות

הערה: פרשן הסקריפטים רץ בסביבת sandbox מאובטחת ללא גישה למערכת הקבצים או לרשת. לא ניתן להריץ קוד זדוני.

פונקציות ופרוצדורות

חשוב: פונקציות (FUNCTION) ופרוצדורות (SUB) חייבות להיות מוגדרות לפני הקריאה הראשונה. הפרשן קורא את הקוד מלמעלה למטה: פונקציה או פרוצדורה ידועה רק אחרי שהגדרתה עובדה.

SUB (פרוצדורה ללא ערך החזרה)

SUB calcular(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (פונקציה עם ערך החזרה)

FUNCTION duplicar(x)
    duplicar = x * 2
ENDFUNCTION

DIM resultado
resultado = duplicar(21)
OUTPUT 1, resultado

דוגמה: חישוב תוספות

מחשב תוספות לעבודת לילה, ימי חג וסופי שבוע עם טיפול נכון במשמרות שחוצות את חצות (חלוקה לקטעים).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' חישוב שעות לילה (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' שבת = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' ראשון = 7 (ISO-8601)

    ' תעריף גבוה יותר לשעות לילה
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' תעריף גבוה יותר לשעות יום
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' חישוב יום השבוע של היום הבא (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' משמרת שחוצה את חצות - 2 קטעים
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' משמרת יום רגילה
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

הסבר

  • חלוקה לקטעים: עבור משמרות שחוצות את חצות (למשל, 22:00-06:00), המשמרת מחולקת לשני קטעים
  • CalcSegment: מחשב את התוספת עבור קטע תוך התחשבות בתוספות לילה, חג וסוף שבוע
  • תעריפים נפרדים לשבת/ראשון: sarate ו-sorate מאפשרים תוספות שונות עבור שבת וראשון
  • תוספת גבוהה יותר: תמיד מיושמת התוספת הגבוהה ביותר הישימה
  • holidaynextday: מתחשב בכך שהיום הבא הוא חג (חשוב למשמרות לילה)
