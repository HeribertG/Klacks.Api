Helgdagsregler - Förklaringar

Regelformat

Det finns två typer av regler: fasta datum och påskrelaterade datum.

Fasta datum

Format: MM/DD

ExempelBetydelse
01/011 januari (nyårsdagen)
12/2525 december (juldagen)
08/011 augusti (schweiziska nationaldagen)

Fasta datum med veckodagsförskjutning

Format: MM/DD+XX+WW eller MM/DD+XX-WW

  • +XX = tecken + förskjutning i dagar (00 till 99)
  • + = nästa veckodag efter datumet
  • - = föregående veckodag före datumet
  • WW = veckodag (SU, MO, TU, WE, TH, FR, SA)
ExempelBetydelse
11/01+00+THFörsta torsdagen i november
11/22+00+THFjärde torsdagen i november (Thanksgiving)
05/25+00-MOSista måndagen i maj (Memorial Day)

Påskrelaterade datum

Format: EASTER+XX eller EASTER-XX

ExempelBetydelse
EASTER+00Påskdagen
EASTER-02Långfredagen (2 dagar före påsk)
EASTER+01Annandag påsk (1 dag efter påsk)
EASTER+39Kristi himmelsfärdsdag (39 dagar efter påsk)
EASTER+49Pingstdagen (49 dagar efter påsk)
EASTER+50Annandag pingst (50 dagar efter påsk)
EASTER+60Kristi kropps och blods högtid (60 dagar efter påsk)

Underregelsformat

Underregler möjliggör automatisk förskjutning av en helgdag när den infaller på en viss veckodag.

Format: WW+X eller WW-X, flera regler separeras med ;

  • WW = veckodag (SU, MO, TU, WE, TH, FR, SA)
  • +X = flytta X dagar framåt
  • -X = flytta X dagar bakåt
ExempelBetydelse
SA+2;SU+1Om lördag: +2 dagar (måndag), om söndag: +1 dag (måndag)
SU+1Om söndag: flytta till måndag

Obs: Underregler används vanligtvis för att flytta helgdagar som infaller på helgen till nästa vardag.
