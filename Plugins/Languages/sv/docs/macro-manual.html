Handbok för makroskript

Inledning

Makroredigeraren gör det möjligt att skriva skript som används som beräkningsunderlag för tjänster, arbetsaktiviteter m.m.

Användningsexempel:

  • Exakt timberäkning för semester eller militärtjänst baserat på sysselsättningsgrad
  • Beräkning av tillägg för nattarbete, helger och helgdagar
  • Flexibel anpassning till nya krav

Obs: Skripttolken körs i en säker sandlådemiljö utan åtkomst till filsystemet eller nätverket. Det är inte möjligt att köra skadlig kod.

Kontrollstrukturer

Viktigt: Nyckelord som ENDIF, ENDFUNCTION, ENDSUB måste skrivas ihop (inte END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "stor"
ELSEIF x > 5 THEN
    OUTPUT 1, "medelstor"
ELSE
    OUTPUT 1, "liten"
ENDIF

IF på en rad

IF x > 10 THEN OUTPUT 1, "stor" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Helg"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Vardag"
    CASE ELSE
        OUTPUT 1, "Okänd"
END SELECT

FOR-NEXT-loop

FOR i = 1 TO 10
    summa += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

DO-LOOP-loop

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Funktioner och procedurer

Viktigt: Funktioner (FUNCTION) och procedurer (SUB) måste definieras före det första anropet. Tolken läser koden uppifrån och ner: en funktion eller procedur är först känd efter att dess definition har bearbetats.

SUB (procedur utan returvärde)

SUB beräkna(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (funktion med returvärde)

FUNCTION fördubbla(x)
    fördubbla = x * 2
ENDFUNCTION

DIM resultat
resultat = fördubbla(21)
OUTPUT 1, resultat

Exempel: Tilläggsberäkning

Beräknar tillägg för nattarbete, helgdagar och helger med korrekt hantering av skift som passerar midnatt (segmentindelning).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Beräkna nattimmar (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Lö = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Sö = 7 (ISO-8601)

    ' Högsta sats för nattimmar
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Högsta sats för dagtimmar
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Beräkna veckodagen för nästa dag (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Skift som passerar midnatt - 2 segment
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Normalt dagskift
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Förklaring

  • Segmentindelning: För skift som passerar midnatt (t.ex. 22:00-06:00) delas skiftet i två segment
  • CalcSegment: Beräknar tillägget för ett segment med hänsyn till natt-, helgdags- och helgtillägg
  • Separata satser för lördag/söndag: sarate och sorate tillåter olika tillägg för lördag och söndag
  • Högsta tillägg: Det högsta tillämpliga tillägget används alltid
  • holidaynextday: Tar hänsyn till om nästa dag är en helgdag (viktigt för nattskift)
