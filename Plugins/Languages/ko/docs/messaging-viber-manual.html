Viber 봇 설정

사전 준비

Viber를 통해 메시지를 보내고 받으려면 Viber Bot Account가 필요합니다. Viber는 동유럽과 중동에서 특히 인기 있습니다.

참고: Viber를 통한 서비스 메시지에는 요금이 부과됩니다. 자세한 정보는 partners.viber.com을 방문하세요.

1단계 — Viber Bot Account 생성

  1. partners.viber.com에서 가입합니다.
  2. Viber 관리자 패널에서 새 Bot Account를 생성합니다.
  3. 생성 후 할당된 Auth Token을 확인합니다.
  4. 발신자 이름을 설정합니다 (예: Klacks Bot).

2단계 — Klacks에서 Viber 구성

  1. 설정 → 메시징 프로바이더로 이동합니다.
  2. 추가를 클릭합니다.
  3. 다음과 같이 필드를 입력합니다:
필드설명예
이름고유한 내부 이름viber-bot
표시 이름사용자 인터페이스에 표시됨Viber
프로바이더 유형Viber 선택Viber
Auth TokenViber 관리자 패널의 Auth Token4a7f8b2c...
발신자 이름수신자에게 표시되는 이름Klacks Bot
활성화프로바이더 활성화/비활성화예
  1. 저장을 클릭합니다.
  2. 프로바이더 목록의 테스트 버튼으로 연결을 테스트합니다.

3단계 — 챗봇(Klacksy)을 통한 메시지 전송

Viber 프로바이더가 구성되고 활성화되면 챗봇 Klacksy가 메시지를 보낼 수 있습니다. 전화번호를 알 필요 없이 직원 또는 고객의 이름만 사용하면 됩니다: 

"홍길동에게 메시지 보내기: 내일 오후 2시에 예약이 있습니다."
"김영희에게 메시지 보내기: 근무가 교대되었습니다."
"이철수에게 월요일 근무가 취소되었다고 알려주세요."

Klacksy가 자동으로 수신자의 휴대전화 번호를 조회하고 구성된 Viber 프로바이더를 사용합니다. 이 과정은 사용자에게 완전히 투명합니다 — 이름으로 작업하면 Klacksy가 나머지를 처리합니다.

전제 조건: 수신자의 휴대전화 번호가 Klacks에 저장되어 있어야 합니다 (주소 → 연락처 정보).

4단계 — 수신 메시지 (Webhook)

수신 Viber 메시지를 Klacks에 표시하려면 Viber 관리자 패널에서 Webhook을 구성해야 합니다.

  1. 설정 → 메시징 프로바이더로 이동합니다.
  2. Viber 프로바이더의 자동 생성된 Webhook 시크릿을 확인합니다.
  3. Viber 관리자 패널에서 Webhook URL을 구성합니다: 
    https://{your-domain}/api/messaging/webhook/viber
  4. 수신 메시지가 이제 메시징 수신함에 실시간으로 표시됩니다.

문제 해결

문제가능한 원인해결 방법
연결 테스트 실패 Auth Token이 잘못됨 Viber 관리자 패널에서 토큰을 확인하세요
메시지가 전달되지 않음 수신자에게 Viber 계정이 없음 수신자가 Viber를 사용하고 있는지 확인하세요
수신자가 메시지를 받지 못함 휴대전화 번호가 올바르게 저장되지 않음 연락처 정보에서 휴대전화 번호를 확인하세요
수신 메시지 누락 Webhook이 구성되지 않음 Viber 관리자 패널에서 Webhook URL을 확인하세요
계정 잔액 소진 서비스 메시지 예산 소진 Viber 관리자 패널에서 잔액을 충전하세요
