매크로 스크립팅 매뉴얼

소개

매크로 편집기를 사용하여 업무, 고용 형태 등의 계산 기반으로 사용되는 스크립트를 작성할 수 있습니다.

사용 예:

  • 고용률에 기반한 휴가 및 군복무 시간의 정확한 계산
  • 야간 근무, 주말, 공휴일 할증 수당 계산
  • 새로운 요구사항에 대한 유연한 대응

참고: 스크립트 인터프리터는 파일 시스템이나 네트워크에 접근할 수 없는 안전한 샌드박스 환경에서 동작합니다. 악성 코드를 실행할 수 없습니다.

변수와 상수

모든 변수와 상수는 타입이 없습니다.

CONST PI = 3.1415
DIM a, b, c
IMPORT betrag, rabatt

가져오기 변수 (IMPORT)

이 변수들은 Work, Contract, CalendarSelection에서 생성됩니다:

변수타입설명
hourDecimal근무 시간 (Work에서 가져옴)
fromhourDecimal시작 시각 (10진 시간) (8:30 → 8.5)
untilhourDecimal종료 시각 (10진 시간) (17:00 → 17.0)
weekdayInteger요일 ISO-8601 (1=월, 2=화, 3=수, 4=목, 5=금, 6=토, 7=일)
holidayBoolean해당일이 공식 공휴일인지 여부
holidaynextdayBoolean다음 날이 공식 공휴일인지 여부
nightrateDecimalContract의 야간 할증률 (예: 0.10 = 10%)
holidayrateDecimalContract의 공휴일 할증률 (예: 0.15 = 15%)
sarateDecimalContract의 토요일 할증률
sorateDecimalContract의 일요일 할증률
guaranteedhoursDecimalContract의 보장 월간 시간 수
fulltimeDecimalContract의 풀타임 시간 수

중요: ENDIF, ENDFUNCTION, ENDSUB 등의 키워드는 이어서 기술해야 합니다 (END IF처럼 분리하여 기술하지 마세요).

    OUTPUT 1, "큼"

함수와 프로시저

중요: 함수(FUNCTION)와 프로시저(SUB)는 최초 호출보다 앞에 정의해야 합니다. 인터프리터는 코드를 위에서 아래로 읽습니다. 함수 또는 프로시저는 정의가 처리된 후에만 사용할 수 있습니다.

SUB (반환값 없는 프로시저)

SUB berechne(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (반환값 있는 함수)

FUNCTION verdoppeln(x)
    verdoppeln = x * 2
ENDFUNCTION

DIM ergebnis
ergebnis = verdoppeln(21)
OUTPUT 1, ergebnis

예: 할증 수당 계산

자정을 넘는 교대근무를 올바르게 처리한 (세그먼트 분할) 야간, 공휴일, 주말 할증 수당을 계산합니다.

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' 야간 시간 계산 (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' 토 = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' 일 = 7 (ISO-8601)

    ' 야간 시간의 최고 할증률
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' 주간 시간의 최고 할증률
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' 다음 날의 요일 계산 (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' 자정을 넘는 교대근무 - 2개 세그먼트
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' 일반 주간 교대근무
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

설명

  • 세그먼트 분할: 자정을 넘는 교대근무 (예: 22:00-06:00)의 경우 교대를 2개 세그먼트로 분할합니다
  • CalcSegment: 야간, 공휴일, 주말 할증 수당을 고려하여 세그먼트의 할증액을 계산합니다
  • 토/일 별도 할증률: sarate와 sorate를 통해 토요일과 일요일에 다른 할증률을 설정할 수 있습니다
  • 최고 할증률: 항상 적용 가능한 최고의 할증률이 사용됩니다
  • holidaynextday: 다음 날이 공휴일인지 고려합니다 (야간 교대근무에 중요)
