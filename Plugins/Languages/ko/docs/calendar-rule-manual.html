규칙 형식

규칙에는 2가지 유형이 있습니다: 고정 날짜와 부활절 기준 날짜입니다.

고정 날짜

형식: MM/DD

예의미
01/011월 1일 (새해)
12/2512월 25일 (크리스마스)
08/018월 1일 (스위스 건국기념일)

요일 이동이 있는 고정 날짜

형식: MM/DD+XX+WW 또는 MM/DD+XX-WW

  • +XX = 부호 + 오프셋(일수) (00부터 99)
  • + = 날짜 이후의 다음 요일
  • - = 날짜 이전의 이전 요일
  • WW = 요일 (SU, MO, TU, WE, TH, FR, SA)
예의미
11/01+00+TH11월 첫 번째 목요일
11/22+00+TH11월 네 번째 목요일 (추수감사절)
05/25+00-MO5월 마지막 월요일 (메모리얼 데이)

부활절 기준 날짜

형식: EASTER+XX 또는 EASTER-XX

예의미
EASTER+00부활절 일요일
EASTER-02성금요일 (부활절 2일 전)
EASTER+01부활절 월요일 (부활절 1일 후)
EASTER+39승천일 (부활절 39일 후)
EASTER+49성령강림절 (부활절 49일 후)
EASTER+50성령강림절 다음 날 (부활절 50일 후)
EASTER+60성체축일 (부활절 60일 후)

하위규칙 형식

하위규칙은 공휴일이 특정 요일에 해당할 경우 자동으로 대체일을 지정할 수 있게 합니다.

형식: WW+X 또는 WW-X, 여러 규칙은 ;로 구분합니다

  • WW = 요일 (SU, MO, TU, WE, TH, FR, SA)
  • +X = X일 뒤로 이동
  • -X = X일 앞으로 이동
예의미
SA+2;SU+1토요일인 경우: +2일(월요일), 일요일인 경우: +1일(월요일)
SU+1일요일인 경우: 월요일로 대체

참고: 하위규칙은 보통 주말에 해당하는 공휴일을 다음 평일로 대체하는 데 사용됩니다.

요일 약어

약어요일
SU일요일 (Sunday)
MO월요일 (Monday)
TU화요일 (Tuesday)
WE수요일 (Wednesday)
TH목요일 (Thursday)
