SMS 프로바이더 설정

사전 준비

Klacks를 통해 SMS 메시지를 보내고 받으려면 SMS 게이트웨이 프로바이더 계정이 필요합니다. Klacks는 REST API를 제공하는 모든 프로바이더를 지원합니다. 예:

  • Twilio — 글로벌 서비스, 사용량 기반 과금
  • Vonage (Nexmo) — 유럽 프로바이더, GDPR 준수
  • MessageBird — 네덜란드 프로바이더, EU 데이터 센터

참고: 아래 단계는 Twilio를 예시로 사용합니다. 다른 프로바이더는 다른 용어를 사용할 수 있지만 원칙은 동일합니다.

1단계 — SMS 프로바이더에서 계정 생성

  1. 원하는 프로바이더에 가입합니다 (예: twilio.com).
  2. 전화번호와 이메일 주소를 인증합니다.
  3. 발신 번호를 구매합니다. 이 번호가 발신 SMS의 발신자로 표시됩니다.
  4. 프로바이더 대시보드에서 API 키 (API Key / Auth Token)를 확인합니다.
  5. 프로바이더의 게이트웨이 URL (API 엔드포인트)를 확인합니다.
프로바이더게이트웨이 URL
Twiliohttps://api.twilio.com/2010-04-01/Accounts/{AccountSID}/Messages.json
Vonagehttps://rest.nexmo.com/sms/json
MessageBirdhttps://rest.messagebird.com/messages

2단계 — Klacks에서 SMS 프로바이더 구성

  1. 설정 → 메시징 프로바이더로 이동합니다.
  2. 추가를 클릭합니다.
  3. 다음과 같이 필드를 입력합니다:
필드설명예
이름고유한 내부 이름sms-twilio
표시 이름사용자 인터페이스에 표시됨SMS (Twilio)
프로바이더 유형SMS 선택SMS
API 키프로바이더의 Auth Token / API Keysk_live_abc123...
발신 번호구매한 전화번호 (국제 형식)+41791234567
게이트웨이 URLSMS 프로바이더의 API 엔드포인트https://api.twilio.com/...
활성화프로바이더 활성화/비활성화예
  1. 저장을 클릭합니다.
  2. 프로바이더 목록의 테스트 버튼으로 연결을 테스트합니다.

3단계 — 챗봇(Klacksy)을 통한 메시지 전송

SMS 프로바이더가 구성되고 활성화되면 챗봇 Klacksy가 메시지를 보낼 수 있습니다. 전화번호를 알 필요 없이 직원 또는 고객의 이름만 사용하면 됩니다: 

"홍길동에게 메시지 보내기: 내일 오후 2시에 예약이 있습니다."
"김영희에게 메시지 보내기: 근무가 교대되었습니다."
"이철수에게 월요일 근무가 취소되었다고 알려주세요."

Klacksy가 자동으로 수신자의 전화번호를 조회하고 구성된 SMS 프로바이더를 사용합니다. 이 과정은 사용자에게 완전히 투명합니다 — 이름으로 작업하면 Klacksy가 나머지를 처리합니다.

전제 조건: 수신자의 휴대전화 번호가 Klacks에 저장되어 있어야 합니다 (주소 → 연락처 정보).

4단계 — 수신 SMS (Webhook)

수신 SMS를 Klacks에 표시하려면 SMS 프로바이더에서 Webhook을 구성해야 합니다.

  1. 설정 → 메시징 프로바이더로 이동합니다.
  2. SMS 프로바이더의 자동 생성된 Webhook 시크릿을 확인합니다.
  3. SMS 프로바이더 대시보드에서 Webhook URL을 구성합니다: 
    https://{your-domain}/api/messaging/webhook/sms
  4. 수신 SMS가 이제 메시징 수신함에 실시간으로 표시됩니다.

문제 해결

문제가능한 원인해결 방법
연결 테스트 실패 API 키 또는 게이트웨이 URL이 잘못됨 프로바이더 대시보드에서 자격 증명을 확인하세요
SMS가 전달되지 않음 발신 번호가 인증되지 않음 프로바이더에서 번호가 활성화되어 있는지 확인하세요
수신자가 SMS를 받지 못함 전화번호가 국제 형식이 아님 +{국가코드}{번호} 형식을 사용하세요 (예: +41791234567)
수신 SMS 누락 Webhook이 구성되지 않음 프로바이더 대시보드에서 Webhook URL을 확인하세요
계정 잔액 소진 사용량 기반 과금 예산 소진 프로바이더에서 잔액을 충전하세요
