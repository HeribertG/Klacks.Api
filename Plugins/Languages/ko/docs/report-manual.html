보고서 디자이너 매뉴얼

보고서 구조

보고서는 3개의 섹션으로 구성됩니다:

  • 헤더 – 각 페이지 상단에 인쇄됩니다
  • 본문 – 데이터 테이블을 포함합니다
  • 푸터 – 각 페이지 하단에 인쇄됩니다

각 섹션은 보고서 편집기의 디자인 탭에서 편집할 수 있습니다.

헤더 섹션

헤더에는 3개의 영역이 있습니다: 왼쪽, 가운데, 오른쪽.

각 영역에는 다음을 포함할 수 있습니다:

  • 필드 – 자동으로 입력되는 데이터베이스 필드
  • 자유 텍스트 – 수동으로 입력한 텍스트
  • 이미지 – 예: 회사 로고 (이미지 관리자에서 업로드)

행은 도구 모음에서 추가하거나 삭제할 수 있습니다.

본문 테이블

본문에는 하나 이상의 테이블이 포함됩니다.

작업설명
열 추가+ 아이콘 또는 필드 목록에서 드래그 앤 드롭으로 추가
열 너비열 경계선을 드래그하여 조정
열 순서드래그 앤 드롭으로 이동
열 삭제열 편집기의 삭제 아이콘으로 삭제

각 테이블은 데이터셋에 연결되어 있습니다 (예: 근무 일정, 주소).

필드 결합 (병합 필드)

병합 필드를 사용하면 하나의 열에 여러 데이터 바인딩을 설정할 수 있습니다.

  • 열 편집기에서 필드 추가를 클릭하여 같은 열에 추가 필드를 추가합니다
  • 값은 세로로 나열되거나 구분 기호를 사이에 두고 표시됩니다
  • 구분 기호는 필드 대화 상자에서 설정할 수 있습니다 (예: 쉼표, 공백, 줄 바꿈)

테이블 푸터

각 테이블 아래에 푸터를 표시할 수 있습니다.

  • SUM 필드: 숫자 열을 자동으로 합산할 수 있습니다
  • 레이블: 합계의 레이블을 표시하거나 숨길 수 있습니다
  • 열 편집기의 푸터 탭에서 활성화합니다

자유 텍스트 행

자유 텍스트 행은 데이터 테이블 앞 또는 뒤에 삽입할 수 있습니다.

속성설명
위치데이터 앞 또는 데이터 뒤를 선택 가능
내용임의의 텍스트, 예: 주의사항, 조건, 서명란
스타일글꼴, 크기, 정렬을 설정 가능

서식 설정

각 셀 및 자유 텍스트를 개별적으로 서식 설정할 수 있습니다:

옵션설명
글꼴셀에 선택 가능한 글꼴
글꼴 크기포인트(pt) 단위 크기
굵게 / 기울임 / 밑줄B / I / U 버튼으로 전환
글자 색상자유 선택 가능한 색상
정렬왼쪽 정렬, 가운데 정렬, 또는 오른쪽 정렬

셀 테두리

각 셀의 테두리를 개별적으로 설정할 수 있습니다.

속성옵션
변위, 오른쪽, 아래, 왼쪽 (개별 또는 전체)
선 스타일가는 선, 중간 선, 굵은 선, 파선, 이중선
선 색상자유 선택 가능

테두리는 열 편집기의 테두리에서 설정합니다.

페이지 설정

일반 탭에서 인쇄 설정을 조정할 수 있습니다:

  • 방향: 세로 (포트레이트) 또는 가로 (랜드스케이프)
  • 용지 크기: A4, A3, B4(JIS), B5(JIS) 또는 레터
  • 페이지 여백: 위, 오른쪽, 아래, 왼쪽 (mm 단위)

표시 옵션 (디자이너)

데이터 소스 근무 일정(Schedule)의 경우 디자인 탭에 추가 옵션이 표시됩니다:

  • 같은 날의 항목을 합치기: 같은 날의 여러 항목을 1행으로 합쳐서 표시합니다
  • 전체 기간 표시: 항목이 없는 날도 포함하여 기간 내 모든 날을 표시합니다

데이터 소스

보고서에서 사용 가능한 데이터 소스:

데이터 소스설명
근무 일정(Schedule)교대 및 근무 시간
주소 (목록)모든 주소를 테이블 형식으로 표시
주소 (상세)개별 주소
부재부재 목록
교대(Shifts)교대 테이블
그룹그룹 목록
컨테이너 템플릿컨테이너 보고서용 템플릿

데이터 소스는 "데이터 소스" 탭에서 선택하며 사용 가능한 필드가 결정됩니다.
