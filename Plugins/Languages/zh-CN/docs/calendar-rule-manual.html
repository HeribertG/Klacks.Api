日历规则 - 说明

规则格式

规则分为两种类型：固定日期和基于复活节的日期。

固定日期

格式：MM/DD

示例含义
01/011月1日（元旦）
12/2512月25日（圣诞节）
08/018月1日（瑞士国庆日）

带星期偏移的固定日期

格式：MM/DD+XX+WW 或 MM/DD+XX-WW

  • +XX = 符号 + 偏移量（天数）（00至99）
  • + = 该日期之后的下一个星期几
  • - = 该日期之前的上一个星期几
  • WW = 星期几（SU、MO、TU、WE、TH、FR、SA）
示例含义
11/01+00+TH11月的第1个星期四
11/22+00+TH11月的第4个星期四（感恩节）
05/25+00-MO5月最后一个星期一（阵亡将士纪念日）

基于复活节的日期

格式：EASTER+XX 或 EASTER-XX

示例含义
EASTER+00复活节星期日
EASTER-02耶稣受难日（复活节前2天）
EASTER+01复活节星期一（复活节后1天）
EASTER+39耶稣升天节（复活节后39天）
EASTER+49五旬节（复活节后49天）
EASTER+50圣灵降临节次日（复活节后50天）
EASTER+60基督圣体节（复活节后60天）

子规则格式

子规则允许在节假日落在特定星期几时自动进行调休。

格式：WW+X 或 WW-X，多个规则用 ; 分隔

  • WW = 星期几（SU、MO、TU、WE、TH、FR、SA）
  • +X = 向后移动X天
  • -X = 向前移动X天
示例含义
SA+2;SU+1如遇星期六：+2天（星期一）；如遇星期日：+1天（星期一）
SU+1如遇星期日：调休至星期一

注意：子规则通常用于将落在周末的节假日调休至下一个工作日。

星期几缩写

缩写星期
SU星期日（Sunday）
MO星期一（Monday）
TU星期二（Tuesday）
WE星期三（Wednesday）
TH星期四（Thursday）
FR星期五（Friday）
SA星期六（Saturday）

复选框

复选框含义
法定节假日该日为官方法定节假日
带薪该节假日为带薪假（与薪资计算相关）
