Juhlapyhäsäännöt - Selitykset

Sääntömuoto

Sääntöjä on kahdenlaisia: kiinteät päivämäärät ja pääsiäiseen sidotut päivämäärät.

Kiinteät päivämäärät

Muoto: MM/DD

EsimerkkiMerkitys
01/011. tammikuuta (uudenvuodenpäivä)
12/2525. joulukuuta (joulupäivä)
08/011. elokuuta (Sveitsin kansallispäivä)

Kiinteät päivämäärät viikonpäiväsiirrolla

Muoto: MM/DD+XX+WW tai MM/DD+XX-WW

  • +XX = merkki + siirros päivissä (00-99)
  • + = seuraava viikonpäivä päivämäärän jälkeen
  • - = edellinen viikonpäivä ennen päivämäärää
  • WW = viikonpäivä (SU, MO, TU, WE, TH, FR, SA)
EsimerkkiMerkitys
11/01+00+THMarraskuun ensimmäinen torstai
11/22+00+THMarraskuun neljäs torstai (kiitospäivä)
05/25+00-MOToukokuun viimeinen maanantai (muistopäivä)

Alisääntöjen muoto

Alisäännöt mahdollistavat juhlapyhän automaattisen siirtämisen, kun se osuu tietylle viikonpäivälle.

Muoto: WW+X tai WW-X, useita sääntöjä erotetaan ;-merkillä

  • WW = viikonpäivä (SU, MO, TU, WE, TH, FR, SA)
  • +X = siirrä X päivää eteenpäin
  • -X = siirrä X päivää taaksepäin
EsimerkkiMerkitys
SA+2;SU+1Jos lauantai: +2 päivää (maanantai), jos sunnuntai: +1 päivä (maanantai)
SU+1Jos sunnuntai: siirrä maanantaille

Huomautus: Alisääntöjä käytetään tyypillisesti siirtämään viikonloppuun osuvat juhlapyhät seuraavalle arkipäivälle.
