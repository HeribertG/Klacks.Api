Makroskriptauksen käsikirja

Johdanto

Makroeditori mahdollistaa skriptien kirjoittamisen, joita käytetään laskentaperusteena palveluille, työtoiminnoille jne.

Käyttöesimerkkejä:

  • Tarkka tuntilaskenta lomille tai asepalvelukselle työsuhteen asteen mukaan
  • Yötyön, viikonloppujen ja juhlapyhien lisien laskenta
  • Joustava mukautuminen uusiin vaatimuksiin

Huomautus: Skriptitulkki toimii turvallisessa hiekkalaatikkoympäristössä ilman pääsyä tiedostojärjestelmään tai verkkoon. Haitallisen koodin suorittaminen ei ole mahdollista.

Ohjausrakenteet

Tärkeää: Avainsanat kuten ENDIF, ENDFUNCTION, ENDSUB on kirjoitettava yhteen (ei END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "suuri"
ELSEIF x > 5 THEN
    OUTPUT 1, "keskikokoinen"
ELSE
    OUTPUT 1, "pieni"
ENDIF

IF yhdellä rivillä

IF x > 10 THEN OUTPUT 1, "suuri" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Viikonloppu"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Arkipäivä"
    CASE ELSE
        OUTPUT 1, "Tuntematon"
END SELECT

FOR-NEXT-silmukka

FOR i = 1 TO 10
    summa += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

DO-LOOP-silmukka

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Funktiot ja proseduurit

Tärkeää: Funktiot (FUNCTION) ja proseduurit (SUB) on määriteltävä ennen ensimmäistä kutsua. Tulkki lukee koodin ylhäältä alas: funktio tai proseduuri tunnetaan vasta kun sen määritelmä on käsitelty.

SUB (proseduuri ilman paluuarvoa)

SUB laske(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (funktio paluuarvolla)

FUNCTION tuplaa(x)
    tuplaa = x * 2
ENDFUNCTION

DIM tulos
tulos = tuplaa(21)
OUTPUT 1, tulos

Esimerkki: Lisien laskenta

Laskee lisät yötyöstä, juhlapyhistä ja viikonlopuista oikealla käsittelyllä keskiyön ylittäville vuoroille (segmenttijako).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Laske yötunnit (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' La = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Su = 7 (ISO-8601)

    ' Korkein taso yötunneille
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Korkein taso päivätunneille
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Laske seuraavan päivän viikonpäivä (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Keskiyön ylittävä vuoro - 2 segmenttiä
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Normaali päivävuoro
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Selitys

  • Segmenttijako: Keskiyön ylittäville vuoroille (esim. 22:00-06:00) vuoro jaetaan kahteen segmenttiin
  • CalcSegment: Laskee lisän segmentille huomioiden yö-, juhlapyhä- ja viikonloppulisät
  • Erilliset tasot lauantaille/sunnuntaille: sarate ja sorate mahdollistavat erilaiset lisät lauantaille ja sunnuntaille
  • Korkein lisä: Aina sovelletaan korkeinta voimassa olevaa lisää
  • holidaynextday: Huomioi onko seuraava päivä juhlapyhä (tärkeää yövuoroille)
