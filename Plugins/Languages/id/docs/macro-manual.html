Panduan Skrip Makro

Pendahuluan

Dengan editor makro, Anda dapat membuat skrip yang digunakan sebagai dasar perhitungan untuk pekerjaan, jenis kontrak kerja, dan lainnya.

Contoh penggunaan:

  • Perhitungan akurat waktu cuti dan dinas militer berdasarkan tingkat pekerjaan
  • Perhitungan upah lembur untuk kerja malam, akhir pekan, dan hari libur
  • Penyesuaian fleksibel terhadap persyaratan baru

Catatan: Interpreter skrip berjalan di lingkungan sandbox yang aman tanpa akses ke sistem file atau jaringan. Kode berbahaya tidak dapat dijalankan.

Struktur Kontrol

Penting: Kata kunci seperti ENDIF, ENDFUNCTION, ENDSUB harus ditulis menyatu (jangan ditulis terpisah seperti END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "Besar"
ELSEIF x > 5 THEN
    OUTPUT 1, "Sedang"
ELSE
    OUTPUT 1, "Kecil"
ENDIF

IF Satu Baris

IF x > 10 THEN OUTPUT 1, "Besar" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Akhir pekan"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Hari kerja"
    CASE ELSE
        OUTPUT 1, "Tidak diketahui"
END SELECT

Loop FOR-NEXT

FOR i = 1 TO 10
    summe += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

Loop DO-LOOP

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Fungsi dan Prosedur

Penting: Fungsi (FUNCTION) dan prosedur (SUB) harus didefinisikan sebelum pemanggilan pertama. Interpreter membaca kode dari atas ke bawah. Fungsi atau prosedur hanya dapat digunakan setelah definisinya diproses.

SUB (Prosedur tanpa nilai kembali)

SUB hitung(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (Fungsi dengan nilai kembali)

FUNCTION gandakan(x)
    gandakan = x * 2
ENDFUNCTION

DIM hasil
hasil = gandakan(21)
OUTPUT 1, hasil

Contoh: Perhitungan Upah Lembur

Menghitung upah lembur malam, hari libur, dan akhir pekan dengan penanganan shift yang melewati tengah malam (pembagian segmen).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Hitung jam malam (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Sab = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Min = 7 (ISO-8601)

    ' Tarif tertinggi untuk jam malam
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Tarif tertinggi untuk jam siang
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Hitung hari berikutnya (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Shift melewati tengah malam - 2 segmen
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Shift siang normal
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Penjelasan

  • Pembagian segmen: Untuk shift yang melewati tengah malam (misalnya 22:00-06:00), shift dibagi menjadi 2 segmen
  • CalcSegment: Menghitung tunjangan segmen dengan mempertimbangkan tunjangan malam, hari libur, dan akhir pekan
  • Tarif Sabtu/Minggu terpisah: sarate dan sorate memungkinkan tarif tunjangan yang berbeda untuk Sabtu dan Minggu
  • Tarif tertinggi: Tarif tunjangan tertinggi yang berlaku selalu digunakan
  • holidaynextday: Mempertimbangkan apakah hari berikutnya adalah hari libur (penting untuk shift malam)
