Aturan Hari Libur - Penjelasan

Format Aturan

Ada dua jenis aturan: Tanggal tetap dan Tanggal berbasis Paskah.

Tanggal Tetap

Format: MM/DD

ContohArti
01/011 Januari (Tahun Baru)
12/2525 Desember (Natal)
08/011 Agustus (Hari Nasional Swiss)

Tanggal Tetap dengan Pergeseran Hari

Format: MM/DD+XX+WW atau MM/DD+XX-WW

  • +XX = Tanda + Offset (hari) (00 sampai 99)
  • + = Hari berikutnya setelah tanggal
  • - = Hari sebelumnya sebelum tanggal
  • WW = Hari dalam seminggu (SU, MO, TU, WE, TH, FR, SA)
ContohArti
11/01+00+THKamis pertama di bulan November
11/22+00+THKamis keempat di bulan November (Thanksgiving)
05/25+00-MOSenin terakhir di bulan Mei (Memorial Day)

Tanggal Berbasis Paskah

Format: EASTER+XX atau EASTER-XX

ContohArti
EASTER+00Minggu Paskah
EASTER-02Jumat Agung (2 hari sebelum Paskah)
EASTER+01Senin Paskah (1 hari setelah Paskah)
EASTER+39Kenaikan Isa Almasih (39 hari setelah Paskah)
EASTER+49Pentakosta (49 hari setelah Paskah)
EASTER+50Senin Pentakosta (50 hari setelah Paskah)
EASTER+60Corpus Christi (60 hari setelah Paskah)

Format Sub-Aturan

Sub-aturan memungkinkan penggantian otomatis jika hari libur jatuh pada hari tertentu dalam seminggu.

Format: WW+X atau WW-X, beberapa aturan dipisahkan dengan ;

  • WW = Hari dalam seminggu (SU, MO, TU, WE, TH, FR, SA)
  • +X = Pindahkan X hari ke depan
  • -X = Pindahkan X hari ke belakang
ContohArti
SA+2;SU+1Jika Sabtu: +2 hari (Senin), jika Minggu: +1 hari (Senin)
SU+1Jika Minggu: pindahkan ke Senin

Catatan: Sub-aturan biasanya digunakan untuk memindahkan hari libur yang jatuh pada akhir pekan ke hari kerja berikutnya.
