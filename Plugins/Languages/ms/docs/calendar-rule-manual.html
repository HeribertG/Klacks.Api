Peraturan Hari Kelepasan - Penerangan

Format Peraturan

Terdapat 2 jenis peraturan: Tarikh Tetap dan Tarikh Berasaskan Easter.

Tarikh Tetap

Format: MM/DD

ContohMaksud
01/011 Januari (Tahun Baru)
12/2525 Disember (Krismas)
08/011 Ogos (Hari Kebangsaan Switzerland)

Tarikh Tetap dengan Anjakan Hari

Format: MM/DD+XX+WW atau MM/DD+XX-WW

  • +XX = Tanda + Ofset (bilangan hari) (00 hingga 99)
  • + = Hari minggu seterusnya selepas tarikh
  • - = Hari minggu sebelumnya sebelum tarikh
  • WW = Hari minggu (SU, MO, TU, WE, TH, FR, SA)
ContohMaksud
11/01+00+THKhamis pertama bulan November
11/22+00+THKhamis keempat bulan November (Kesyukuran)
05/25+00-MOIsnin terakhir bulan Mei (Hari Peringatan)

Tarikh Berasaskan Easter

Format: EASTER+XX atau EASTER-XX

ContohMaksud
EASTER+00Ahad Easter (Hari Kebangkitan)
EASTER-02Jumaat Agung (2 hari sebelum Easter)
EASTER+01Isnin Easter (1 hari selepas Easter)
EASTER+39Kenaikan (39 hari selepas Easter)
EASTER+49Pentakosta (49 hari selepas Easter)
EASTER+50Isnin Pentakosta (50 hari selepas Easter)
EASTER+60Corpus Christi (60 hari selepas Easter)

Format Sub-Peraturan

Sub-peraturan membolehkan pemindahan automatik apabila hari kelepasan jatuh pada hari tertentu dalam minggu.

Format: WW+X atau WW-X, beberapa peraturan dipisahkan dengan ;

  • WW = Hari minggu (SU, MO, TU, WE, TH, FR, SA)
  • +X = Alih ke hadapan X hari
  • -X = Alih ke belakang X hari
ContohMaksud
SA+2;SU+1Jika Sabtu: +2 hari (Isnin), jika Ahad: +1 hari (Isnin)
SU+1Jika Ahad: pindah ke Isnin

Nota: Sub-peraturan biasanya digunakan untuk memindahkan hari kelepasan yang jatuh pada hujung minggu ke hari bekerja seterusnya.
