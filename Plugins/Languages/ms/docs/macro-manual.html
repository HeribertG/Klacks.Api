Manual Skrip Makro

Pengenalan

Dengan editor makro, anda boleh mencipta skrip yang digunakan sebagai asas pengiraan untuk kerja, jenis pekerjaan dan sebagainya.

Contoh penggunaan:

  • Pengiraan tepat masa cuti dan perkhidmatan tentera berdasarkan kadar pekerjaan
  • Pengiraan elaun syif malam, hujung minggu dan hari kelepasan
  • Penyesuaian fleksibel kepada keperluan baharu

Nota: Penterjemah skrip beroperasi dalam persekitaran kotak pasir yang selamat tanpa akses kepada sistem fail atau rangkaian. Kod berniat jahat tidak dapat dilaksanakan.

Pembolehubah dan Pemalar

Semua pembolehubah dan pemalar adalah tanpa jenis.

CONST PI = 3.1415
DIM a, b, c
IMPORT betrag, rabatt

Struktur Kawalan

Penting: Kata kunci seperti ENDIF, ENDFUNCTION, ENDSUB mesti ditulis bersambung (jangan tulis END IF secara berasingan).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "Besar"
ELSEIF x > 5 THEN
    OUTPUT 1, "Sederhana"
ELSE
    OUTPUT 1, "Kecil"
ENDIF

IF Satu Baris

IF x > 10 THEN OUTPUT 1, "Besar" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Hujung Minggu"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Hari Bekerja"
    CASE ELSE
        OUTPUT 1, "Tidak Diketahui"
END SELECT

Gelung FOR-NEXT

FOR i = 1 TO 10
    summe += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

Gelung DO-LOOP

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Fungsi dan Prosedur

Penting: Fungsi (FUNCTION) dan prosedur (SUB) mesti ditakrifkan sebelum panggilan pertama. Penterjemah membaca kod dari atas ke bawah. Fungsi atau prosedur hanya boleh digunakan selepas takrifannya diproses.

SUB (Prosedur tanpa nilai pulangan)

SUB kira(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (Fungsi dengan nilai pulangan)

FUNCTION gandakan(x)
    gandakan = x * 2
ENDFUNCTION

DIM hasil
hasil = gandakan(21)
OUTPUT 1, hasil

Contoh: Pengiraan Elaun

Mengira elaun malam, hari kelepasan dan hujung minggu dengan pengendalian yang betul bagi syif merentasi tengah malam (pembahagian segmen).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Kira jam malam (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Sab = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Ahd = 7 (ISO-8601)

    ' Kadar elaun tertinggi untuk jam malam
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Kadar elaun tertinggi untuk jam siang
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Kira hari minggu berikutnya (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Syif merentasi tengah malam - 2 segmen
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Syif siang biasa
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Penerangan

  • Pembahagian segmen: Untuk syif merentasi tengah malam (cth: 22:00-06:00), syif dibahagikan kepada 2 segmen
  • CalcSegment: Mengira elaun segmen dengan mengambil kira elaun malam, hari kelepasan dan hujung minggu
  • Kadar Sabtu/Ahad berasingan: sarate dan sorate membolehkan kadar elaun berbeza untuk Sabtu dan Ahad
  • Kadar elaun tertinggi: Kadar elaun tertinggi yang berkenaan sentiasa digunakan
  • holidaynextday: Mengambil kira sama ada hari berikutnya ialah hari kelepasan (penting untuk syif malam)
