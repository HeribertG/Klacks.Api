祝日ルール - 説明

ルール形式

ルールには2種類あります：固定日付とイースター基準日付です。

固定日付

形式：MM/DD

例意味
01/011月1日（元日）
12/2512月25日（クリスマス）
08/018月1日（スイス建国記念日）

曜日シフト付き固定日付

形式：MM/DD+XX+WW または MM/DD+XX-WW

  • +XX = 符号 + オフセット（日数）（00から99）
  • + = 日付以降の次の曜日
  • - = 日付以前の前の曜日
  • WW = 曜日（SU、MO、TU、WE、TH、FR、SA）
例意味
11/01+00+TH11月の第1木曜日
11/22+00+TH11月の第4木曜日（感謝祭）
05/25+00-MO5月最終月曜日（メモリアルデー）

イースター基準日付

形式：EASTER+XX または EASTER-XX

例意味
EASTER+00イースターサンデー（復活祭）
EASTER-02聖金曜日（イースターの2日前）
EASTER+01イースターマンデー（イースターの1日後）
EASTER+39昇天祭（イースターの39日後）
EASTER+49ペンテコステ（イースターの49日後）
EASTER+50聖霊降臨節翌日（イースターの50日後）
EASTER+60聖体節（イースターの60日後）

サブルール形式

サブルールは、祝日が特定の曜日に当たった場合に自動的に振り替えることを可能にします。

形式：WW+X または WW-X、複数のルールは ; で区切ります

  • WW = 曜日（SU、MO、TU、WE、TH、FR、SA）
  • +X = X日後ろへ移動
  • -X = X日前へ移動
例意味
SA+2;SU+1土曜日の場合：+2日（月曜日）、日曜日の場合：+1日（月曜日）
SU+1日曜日の場合：月曜日に振り替え

注意：サブルールは通常、週末に当たる祝日を次の平日に振り替えるために使用されます。
