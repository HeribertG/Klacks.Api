Manual Scripting Macro

Introducere

Cu editorul de macrocomenzi puteți crea scripturi utilizate ca bază de calcul pentru muncă, tipuri de angajare și altele.

Exemple de utilizare:

  • Calcul precis al timpului de vacanță și serviciu militar pe baza ratei de angajare
  • Calcul sporuri pentru muncă de noapte, weekend și sărbători
  • Adaptare flexibilă la cerințe noi

Notă: Interpretorul de scripturi funcționează într-un mediu sandbox securizat fără acces la sistemul de fișiere sau rețea. Nu poate executa cod malițios.

Structuri de control

Important: Cuvintele cheie precum ENDIF, ENDFUNCTION, ENDSUB trebuie scrise fără spațiu (nu END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "mare"
ELSEIF x > 5 THEN
    OUTPUT 1, "mediu"
ELSE
    OUTPUT 1, "mic"
ENDIF

IF pe o singură linie

IF x > 10 THEN OUTPUT 1, "mare" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Weekend"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Zi lucrătoare"
    CASE ELSE
        OUTPUT 1, "Necunoscut"
END SELECT

Buclă FOR-NEXT

FOR i = 1 TO 10
    summe += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

Buclă DO-LOOP

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Funcții și proceduri

Important: Funcțiile (FUNCTION) și procedurile (SUB) trebuie definite înaintea primului apel. Interpretorul citește codul de sus în jos. O funcție sau procedură poate fi utilizată doar după ce definiția sa a fost procesată.

SUB (procedură fără valoare returnată)

SUB berechne(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (funcție cu valoare returnată)

FUNCTION verdoppeln(x)
    verdoppeln = x * 2
ENDFUNCTION

DIM ergebnis
ergebnis = verdoppeln(21)
OUTPUT 1, ergebnis

Exemplu: Calcul sporuri

Calculează sporuri pentru noapte, sărbători și weekend cu gestionarea corectă a turelor care depășesc miezul nopții (segmentare).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Calcul ore de noapte (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Sâ = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Du = 7 (ISO-8601)

    ' Cel mai mare spor pentru orele de noapte
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Cel mai mare spor pentru orele de zi
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Calcul ziua următoare a săptămânii (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Tură care depășește miezul nopții - 2 segmente
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Tură normală de zi
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Explicație

  • Segmentare: Turele care depășesc miezul nopții (ex.: 22:00-06:00) sunt împărțite în 2 segmente
  • CalcSegment: Calculează sporul unui segment ținând cont de noapte, sărbători și weekend
  • Sporuri separate Sâ/Du: sarate și sorate permit sporuri diferite pentru sâmbătă și duminică
  • Cel mai mare spor: Se aplică întotdeauna cel mai mare spor disponibil
  • holidaynextday: Se ține cont dacă ziua următoare este sărbătoare (important pentru turele de noapte)
