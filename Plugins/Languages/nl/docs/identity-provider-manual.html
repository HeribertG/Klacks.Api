Handleiding Identiteitsprovider

Inleiding

Identiteitsproviders maken de authenticatie van gebruikers mogelijk via externe directoryservices zoals LDAP, Active Directory of OAuth2/OpenID Connect.

Ondersteunde providertypen:

  • LDAP - Lightweight Directory Access Protocol
  • Active Directory - Microsoft Active Directory
  • OAuth2 - OAuth 2.0 autorisatieframework
  • OpenID Connect - Identiteitslaag bovenop OAuth 2.0

Klantimport

Wanneer "Gebruiken voor klantimport" is geactiveerd, worden de volgende gegevens geïmporteerd:

  • Achternaam - sn (achternaam) of displayName
  • Voornaam - givenName
  • Titel - title (functietitel)
  • Bedrijf - company
  • Adres - street/streetAddress, postalCode, l (stad), st (provincie), c (land)
  • Geslacht - Op basis van personalTitle (Dhr./Mevr.) in Active Directory

Opmerking: In OpenLDAP is het geslacht standaard "Neutraal" en kan handmatig worden gecorrigeerd.

Tips

Synology NAS als LDAP-server

Een Synology NAS kan als LDAP-server worden gebruikt. Hiervoor moet het pakket "LDAP Server" of een Docker-container met OpenLDAP worden geïnstalleerd.

  • Providertype: "Active Directory" (voor fallback-authenticatie)
  • Gebruikersfilter: (objectClass=inetOrgPerson)
  • Gebruikers-DN-formaat: cn=gebruikersnaam,ou=users,dc=example,dc=com

Verbinding testen

Gebruik de knop "Verbinding testen" om de configuratie te controleren. Bij een succesvolle verbinding worden het aantal gevonden gebruikers en enkele voorbeeldgebruikersnamen weergegeven.
