Handleiding Macro-scripting

Inleiding

De macro-editor maakt het mogelijk scripts te schrijven die als berekeningsbasis dienen voor diensten, werkactiviteiten, enz.

Voorbeelden van gebruik:

  • Exacte urenberekening voor vakantie of militaire dienst op basis van het dienstverband
  • Berekening van toeslagen voor nachtwerk, weekenden en feestdagen
  • Flexibele aanpassing aan nieuwe vereisten

Opmerking: De scriptinterpreter draait in een veilige sandbox-omgeving zonder toegang tot het bestandssysteem of het netwerk. Het is niet mogelijk kwaadaardige code uit te voeren.

Besturingsstructuren

Belangrijk: Sleutelwoorden zoals ENDIF, ENDFUNCTION, ENDSUB moeten aaneengeschreven worden (niet END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "groot"
ELSEIF x > 5 THEN
    OUTPUT 1, "middelgroot"
ELSE
    OUTPUT 1, "klein"
ENDIF

IF op één regel

IF x > 10 THEN OUTPUT 1, "groot" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Weekend"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Werkdag"
    CASE ELSE
        OUTPUT 1, "Onbekend"
END SELECT

FOR-NEXT-lus

FOR i = 1 TO 10
    som += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

DO-LOOP-lus

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Functies en procedures

Belangrijk: Functies (FUNCTION) en procedures (SUB) moeten vóór de eerste aanroep worden gedefinieerd. De interpreter leest de code van boven naar beneden: een functie of procedure is pas bekend nadat de definitie is verwerkt.

SUB (procedure zonder retourwaarde)

SUB berekenen(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (functie met retourwaarde)

FUNCTION verdubbelen(x)
    verdubbelen = x * 2
ENDFUNCTION

DIM resultaat
resultaat = verdubbelen(21)
OUTPUT 1, resultaat

Voorbeeld: Toeslagberekening

Berekent toeslagen voor nachtwerk, feestdagen en weekenden met correcte behandeling van diensten die middernacht overschrijden (segmentdeling).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Nachturen berekenen (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Za = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Zo = 7 (ISO-8601)

    ' Hoogste tarief voor nachturen
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Hoogste tarief voor daguren
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Dag van de week van de volgende dag berekenen (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Dienst die middernacht overschrijdt - 2 segmenten
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Normale dagdienst
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Uitleg

  • Segmentdeling: Voor diensten die middernacht overschrijden (bijv. 22:00-06:00) wordt de dienst in twee segmenten opgedeeld
  • CalcSegment: Berekent de toeslag voor een segment rekening houdend met nacht-, feestdag- en weekendtoeslagen
  • Aparte tarieven voor zaterdag/zondag: sarate en sorate maken verschillende toeslagen voor zaterdag en zondag mogelijk
  • Hoogste toeslag: Altijd wordt de hoogste van toepassing zijnde toeslag gehanteerd
  • holidaynextday: Houdt rekening met of de volgende dag een feestdag is (belangrijk voor nachtdiensten)
