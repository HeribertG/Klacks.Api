กฎมี 2 ประเภท: วันที่คงที่ และ วันที่อิงจากอีสเตอร์

วันที่คงที่

รูปแบบ: MM/DD

ตัวอย่างความหมาย
01/011 มกราคม (วันปีใหม่)
12/2525 ธันวาคม (คริสต์มาส)
08/011 สิงหาคม (วันชาติสวิตเซอร์แลนด์)

วันที่คงที่พร้อมการเลื่อนวัน

รูปแบบ: MM/DD+XX+WW หรือ MM/DD+XX-WW

  • +XX = เครื่องหมาย + ค่าชดเชย (จำนวนวัน) (00 ถึง 99)
  • + = วันถัดไปของสัปดาห์หลังจากวันที่
  • - = วันก่อนหน้าของสัปดาห์ก่อนวันที่
  • WW = วันในสัปดาห์ (SU, MO, TU, WE, TH, FR, SA)
ตัวอย่างความหมาย
11/01+00+THวันพฤหัสบดีแรกของเดือนพฤศจิกายน
11/22+00+THวันพฤหัสบดีที่ 4 ของเดือนพฤศจิกายน (วันขอบคุณพระเจ้า)
05/25+00-MOวันจันทร์สุดท้ายของเดือนพฤษภาคม (วันรำลึก)

วันที่อิงจากอีสเตอร์

รูปแบบ: EASTER+XX หรือ EASTER-XX

ตัวอย่างความหมาย
EASTER+00วันอีสเตอร์ (วันอาทิตย์อีสเตอร์)
EASTER-02วันศุกร์ประเสริฐ (2 วันก่อนอีสเตอร์)
EASTER+01วันจันทร์อีสเตอร์ (1 วันหลังอีสเตอร์)
EASTER+39วันเสด็จขึ้นสวรรค์ (39 วันหลังอีสเตอร์)
EASTER+49วันเพ็นเทคอสต์ (49 วันหลังอีสเตอร์)
EASTER+50วันจันทร์เพ็นเทคอสต์ (50 วันหลังอีสเตอร์)
EASTER+60วันพระกายศักดิ์สิทธิ์ (60 วันหลังอีสเตอร์)

รูปแบบกฎย่อย

กฎย่อยช่วยให้สามารถเลื่อนวันหยุดโดยอัตโนมัติเมื่อตรงกับวันใดวันหนึ่งในสัปดาห์

รูปแบบ: WW+X หรือ WW-X คั่นด้วย ; สำหรับหลายกฎ

  • WW = วันในสัปดาห์ (SU, MO, TU, WE, TH, FR, SA)
  • +X = เลื่อนไปข้างหน้า X วัน
  • -X = เลื่อนไปข้างหลัง X วัน
ตัวอย่างความหมาย
SA+2;SU+1ถ้าตรงวันเสาร์: +2 วัน (วันจันทร์) ถ้าตรงวันอาทิตย์: +1 วัน (วันจันทร์)
SU+1ถ้าตรงวันอาทิตย์: เลื่อนไปวันจันทร์

หมายเหตุ: กฎย่อยมักใช้เพื่อเลื่อนวันหยุดที่ตรงกับวันหยุดสุดสัปดาห์ไปยังวันทำการถัดไป

ตัวย่อวันในสัปดาห์

ตัวย่อวัน
SUวันอาทิตย์ (Sunday)
MOวันจันทร์ (Monday)
TUวันอังคาร (Tuesday)
WEวันพุธ (Wednesday)
THวันพฤหัสบดี (Thursday)
FRวันศุกร์ (Friday)
SAวันเสาร์ (Saturday)
