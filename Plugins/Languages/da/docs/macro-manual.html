Makro-scripting manual

Introduktion

Makro-editoren gør det muligt at skrive scripts, der bruges som beregningsgrundlag for tjenester, arbejdsaktiviteter osv.

Eksempler på anvendelse:

  • Præcis timeberegning for ferie eller militærtjeneste baseret på beskæftigelsesgrad
  • Beregning af tillæg for natarbejde, weekender og helligdage
  • Fleksibel tilpasning til nye krav

Bemærk: Scriptfortolkeren kører i et sikkert sandbox-miljø uden adgang til filsystemet eller netværket. Det er ikke muligt at udføre skadelig kode.

Kontrolstrukturer

Vigtigt: Nøgleord som ENDIF, ENDFUNCTION, ENDSUB skal skrives som ét ord (ikke END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "stor"
ELSEIF x > 5 THEN
    OUTPUT 1, "medium"
ELSE
    OUTPUT 1, "lille"
ENDIF

IF på én linje

IF x > 10 THEN OUTPUT 1, "stor" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Weekend"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Hverdag"
    CASE ELSE
        OUTPUT 1, "Ukendt"
END SELECT

FOR-NEXT-løkke

FOR i = 1 TO 10
    sum += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

DO-LOOP-løkke

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Funktioner og procedurer

Vigtigt: Funktioner (FUNCTION) og procedurer (SUB) skal defineres før det første kald. Fortolkeren læser koden fra top til bund: en funktion eller procedure er først kendt, efter at dens definition er behandlet.

SUB (procedure uden returværdi)

SUB beregn(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (funktion med returværdi)

FUNCTION fordoble(x)
    fordoble = x * 2
ENDFUNCTION

DIM resultat
resultat = fordoble(21)
OUTPUT 1, resultat

Eksempel: Tillægsberegning

Beregner tillæg for natarbejde, helligdage og weekender med korrekt håndtering af vagter, der krydser midnat (segmentopdeling).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Beregn nattimer (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Lø = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Sø = 7 (ISO-8601)

    ' Højeste sats for nattimer
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Højeste sats for dagtimer
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Beregn ugedagen for næste dag (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Vagt der krydser midnat - 2 segmenter
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Normal dagvagt
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Forklaring

  • Segmentopdeling: For vagter der krydser midnat (f.eks. 22:00-06:00) opdeles vagten i to segmenter
  • CalcSegment: Beregner tillægget for et segment under hensyntagen til nat-, helligdags- og weekendtillæg
  • Separate satser for lørdag/søndag: sarate og sorate tillader forskellige tillæg for lørdag og søndag
  • Højeste tillæg: Det højeste gældende tillæg anvendes altid
  • holidaynextday: Tager hensyn til om næste dag er en helligdag (vigtigt for nattevagter)
