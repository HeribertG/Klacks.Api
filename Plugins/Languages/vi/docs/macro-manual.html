Hướng dẫn Scripting Macro

Giới thiệu

Trình soạn thảo macro cho phép viết các script được sử dụng làm cơ sở tính toán cho dịch vụ, hoạt động lao động, v.v.

Ví dụ sử dụng:

  • Tính toán chính xác số giờ cho kỳ nghỉ hoặc nghĩa vụ quân sự theo mức độ làm việc
  • Tính toán phụ cấp cho làm việc ban đêm, cuối tuần và ngày lễ
  • Tùy chỉnh linh hoạt theo các yêu cầu mới

Lưu ý: Trình thông dịch script chạy trong môi trường sandbox an toàn mà không có quyền truy cập hệ thống tệp hay mạng. Không thể thực thi mã độc.

Tất cả các biến và hằng số đều không có kiểu.

Cấu trúc điều khiển

Quan trọng: Các từ khóa như ENDIF, ENDFUNCTION, ENDSUB phải được viết liền nhau (không phải END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "lớn"
ELSEIF x > 5 THEN
    OUTPUT 1, "trung bình"
ELSE
    OUTPUT 1, "nhỏ"
ENDIF

IF trên một dòng

IF x > 10 THEN OUTPUT 1, "lớn" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Cuối tuần"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Ngày làm việc"
    CASE ELSE
        OUTPUT 1, "Không xác định"
END SELECT

Vòng lặp FOR-NEXT

FOR i = 1 TO 10
    suma += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

Vòng lặp DO-LOOP

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Hàm và thủ tục

Quan trọng: Các hàm (FUNCTION) và thủ tục (SUB) phải được định nghĩa trước lần gọi đầu tiên. Trình thông dịch đọc mã từ trên xuống dưới: một hàm hoặc thủ tục chỉ được nhận biết sau khi định nghĩa của nó đã được xử lý.

SUB (thủ tục không có giá trị trả về)

SUB calcular(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (hàm có giá trị trả về)

FUNCTION duplicar(x)
    duplicar = x * 2
ENDFUNCTION

DIM resultado
resultado = duplicar(21)
OUTPUT 1, resultado

Ví dụ: Tính toán phụ cấp

Tính phụ cấp cho làm việc ban đêm, ngày lễ và cuối tuần với việc xử lý chính xác các ca vượt qua nửa đêm (chia thành các phân đoạn).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Tính số giờ ban đêm (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Thứ Bảy = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Chủ Nhật = 7 (ISO-8601)

    ' Tỷ lệ cao nhất cho giờ ban đêm
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Tỷ lệ cao nhất cho giờ ban ngày
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Tính thứ trong tuần của ngày hôm sau (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Ca vượt qua nửa đêm - 2 phân đoạn
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Ca ban ngày bình thường
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Giải thích

  • Chia thành phân đoạn: Đối với các ca vượt qua nửa đêm (ví dụ: 22:00-06:00), ca được chia thành hai phân đoạn
  • CalcSegment: Tính phụ cấp cho một phân đoạn có tính đến phụ cấp ban đêm, ngày lễ và cuối tuần
  • Tỷ lệ riêng cho Thứ Bảy/Chủ Nhật: sarate và sorate cho phép có các mức phụ cấp khác nhau cho Thứ Bảy và Chủ Nhật
  • Phụ cấp cao nhất: Luôn áp dụng mức phụ cấp cao nhất có thể
  • holidaynextday: Tính đến việc ngày hôm sau có phải là ngày lễ hay không (quan trọng đối với các ca ban đêm)
