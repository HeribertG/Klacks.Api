日曆規則 - 說明

規則格式

規則分為兩種類型：固定日期和基於復活節的日期。

固定日期

格式：MM/DD

範例含義
01/011月1日（元旦）
12/2512月25日（聖誕節）
08/018月1日（瑞士國慶日）

帶星期偏移的固定日期

格式：MM/DD+XX+WW 或 MM/DD+XX-WW

  • +XX = 符號 + 偏移量（天數）（00至99）
  • + = 該日期之後的下一個星期幾
  • - = 該日期之前的上一個星期幾
  • WW = 星期幾（SU、MO、TU、WE、TH、FR、SA）
範例含義
11/01+00+TH11月的第1個星期四
11/22+00+TH11月的第4個星期四（感恩節）
05/25+00-MO5月最後一個星期一（陣亡將士紀念日）

基於復活節的日期

格式：EASTER+XX 或 EASTER-XX

範例含義
EASTER+00復活節星期日
EASTER-02耶穌受難日（復活節前2天）
EASTER+01復活節星期一（復活節後1天）
EASTER+39耶穌升天節（復活節後39天）
EASTER+49五旬節（復活節後49天）
EASTER+50聖靈降臨節次日（復活節後50天）
EASTER+60基督聖體節（復活節後60天）

子規則格式

子規則允許在節假日落在特定星期幾時自動進行調休。

格式：WW+X 或 WW-X，多個規則用 ; 分隔

  • WW = 星期幾（SU、MO、TU、WE、TH、FR、SA）
  • +X = 向後移動X天
  • -X = 向前移動X天
範例含義
SA+2;SU+1如遇星期六：+2天（星期一）；如遇星期日：+1天（星期一）
SU+1如遇星期日：調休至星期一

注意：子規則通常用於將落在週末的節假日調休至下一個工作日。

星期幾縮寫

縮寫星期
SU星期日（Sunday）
MO星期一（Monday）
TU星期二（Tuesday）
WE星期三（Wednesday）
TH星期四（Thursday）
FR星期五（Friday）
SA星期六（Saturday）

核取方塊

核取方塊含義
法定節假日該日為官方法定節假日
帶薪該節假日為帶薪假（與薪資計算相關）
