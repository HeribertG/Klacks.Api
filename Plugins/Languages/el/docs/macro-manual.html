Εγχειρίδιο σεναρίων μακροεντολών

Εισαγωγή

Ο επεξεργαστής μακροεντολών επιτρέπει τη συγγραφή σεναρίων που χρησιμοποιούνται ως βάση υπολογισμού για υπηρεσίες, εργασιακές δραστηριότητες κ.λπ.

Παραδείγματα χρήσης:

  • Ακριβής υπολογισμός ωρών για άδεια ή στρατιωτική θητεία βάσει ποσοστού απασχόλησης
  • Υπολογισμός προσαυξήσεων για νυχτερινή εργασία, σαββατοκύριακα και αργίες
  • Ευέλικτη προσαρμογή σε νέες απαιτήσεις

Σημείωση: Ο διερμηνέας σεναρίων εκτελείται σε ασφαλές περιβάλλον sandbox χωρίς πρόσβαση στο σύστημα αρχείων ή στο δίκτυο. Δεν είναι δυνατή η εκτέλεση κακόβουλου κώδικα.

Σημαντικό: Λέξεις-κλειδιά όπως ENDIF, ENDFUNCTION, ENDSUB πρέπει να γράφονται ενωμένα (όχι END IF).

    OUTPUT 1, "μεγάλο"

    OUTPUT 1, "μεσαίο"

Συναρτήσεις και διαδικασίες

Σημαντικό: Οι συναρτήσεις (FUNCTION) και οι διαδικασίες (SUB) πρέπει να ορίζονται πριν από την πρώτη κλήση. Ο διερμηνέας διαβάζει τον κώδικα από πάνω προς τα κάτω: μια συνάρτηση ή διαδικασία είναι γνωστή μόνο μετά την επεξεργασία του ορισμού της.

SUB (διαδικασία χωρίς τιμή επιστροφής)

SUB ypologise(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (συνάρτηση με τιμή επιστροφής)

FUNCTION diplasiazo(x)
    diplasiazo = x * 2
ENDFUNCTION

DIM apotelesma
apotelesma = diplasiazo(21)
OUTPUT 1, apotelesma

Παράδειγμα: Υπολογισμός προσαυξήσεων

Υπολογίζει προσαυξήσεις για νυχτερινή εργασία, αργίες και σαββατοκύριακα με σωστή διαχείριση βαρδιών που υπερβαίνουν τα μεσάνυχτα (τμηματοποίηση).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Υπολογισμός νυχτερινών ωρών (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Σα = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Κυ = 7 (ISO-8601)

    ' Υψηλότερο ποσοστό για νυχτερινές ώρες
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Υψηλότερο ποσοστό για ημερήσιες ώρες
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Υπολογισμός ημέρας εβδομάδας επόμενης ημέρας (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Βάρδια που υπερβαίνει τα μεσάνυχτα - 2 τμήματα
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Κανονική ημερήσια βάρδια
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Επεξήγηση

  • Τμηματοποίηση: Για βάρδιες που υπερβαίνουν τα μεσάνυχτα (π.χ. 22:00-06:00), η βάρδια χωρίζεται σε δύο τμήματα
  • CalcSegment: Υπολογίζει την προσαύξηση για ένα τμήμα λαμβάνοντας υπόψη νυχτερινές, αργιών και σαββατοκύριακου προσαυξήσεις
  • Χωριστά ποσοστά για Σάββατο/Κυριακή: sarate και sorate επιτρέπουν διαφορετικές προσαυξήσεις για Σάββατο και Κυριακή
  • Υψηλότερη προσαύξηση: Εφαρμόζεται πάντα η υψηλότερη ισχύουσα προσαύξηση
  • holidaynextday: Λαμβάνει υπόψη αν η επόμενη ημέρα είναι αργία (σημαντικό για νυχτερινές βάρδιες)
