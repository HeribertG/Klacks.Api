Εγχειρίδιο σχεδιαστή αναφορών

Δομή αναφοράς

Μια αναφορά αποτελείται από τρεις ενότητες:

  • Κεφαλίδα – εκτυπώνεται στο πάνω μέρος κάθε σελίδας
  • Περιεχόμενο – περιέχει τους πίνακες δεδομένων
  • Υποσέλιδο – εκτυπώνεται στο κάτω μέρος κάθε σελίδας

Κάθε ενότητα μπορεί να επεξεργαστεί μέσω της καρτέλας Σχεδίαση στον επεξεργαστή αναφορών.

Ενότητα κεφαλίδας

Η κεφαλίδα έχει τρεις ζώνες: Αριστερά, Κέντρο και Δεξιά.

Κάθε ζώνη μπορεί να περιέχει:

  • Πεδία – πεδία βάσης δεδομένων που συμπληρώνονται αυτόματα
  • Ελεύθερο κείμενο – χειροκίνητα εισαγμένο κείμενο
  • Εικόνες – π.χ. λογότυπο εταιρείας (μεταφόρτωση μέσω του διαχειριστή εικόνων)

Γραμμές μπορούν να προστεθούν ή να αφαιρεθούν μέσω της γραμμής εργαλείων.

Πίνακες περιεχομένου

Το περιεχόμενο περιέχει έναν ή περισσότερους πίνακες.

ΕνέργειαΠεριγραφή
Προσθήκη στήληςΜέσω του συμβόλου + ή σύροντας από τη λίστα πεδίων
Πλάτος στήληςΡύθμιση σύροντας το άκρο της στήλης
Σειρά στηλώνΑλλαγή με σύρσιμο και απόθεση
Διαγραφή στήληςΣτον επεξεργαστή στηλών μέσω του συμβόλου διαγραφής

Κάθε πίνακας συνδέεται με ένα σύνολο δεδομένων (π.χ. πρόγραμμα εργασίας, διευθύνσεις).

Συνδυασμένα πεδία (Merged Fields)

Τα συνδυασμένα πεδία επιτρέπουν πολλαπλές συνδέσεις δεδομένων σε μια μόνο στήλη.

  • Στον επεξεργαστή στηλών, κάντε κλικ στο Προσθήκη πεδίου για να προσθέσετε περισσότερα πεδία στην ίδια στήλη
  • Οι τιμές εμφανίζονται η μία κάτω από την άλλη ή με διαχωριστικό
  • Το διαχωριστικό μπορεί να ρυθμιστεί στο παράθυρο διαλόγου πεδίου (π.χ. κόμμα, κενό, αλλαγή γραμμής)

Υποσέλιδο πίνακα

Κάτω από κάθε πίνακα μπορεί να εμφανίζεται ένα υποσέλιδο.

  • Πεδία SUM: αριθμητικές στήλες μπορούν να αθροίζονται αυτόματα
  • Ετικέτα: η περιγραφή του αθροίσματος μπορεί να εμφανίζεται ή να κρύβεται
  • Ενεργοποιείται στον επεξεργαστή στηλών στην καρτέλα Υποσέλιδο

Γραμμές ελεύθερου κειμένου

Γραμμές ελεύθερου κειμένου μπορούν να εισαχθούν πριν ή μετά τους πίνακες δεδομένων.

ΙδιότηταΠεριγραφή
ΘέσηΕπιλέξιμη Πριν από τα δεδομένα ή Μετά τα δεδομένα
ΠεριεχόμενοΕλεύθερο κείμενο, π.χ. σημειώσεις, όροι ή πεδία υπογραφής
ΣτυλΓραμματοσειρά, μέγεθος και στοίχιση ρυθμιζόμενα

Μορφοποίηση

Κάθε κελί και κάθε ελεύθερο κείμενο μπορεί να μορφοποιηθεί ξεχωριστά:

ΕπιλογήΠεριγραφή
ΓραμματοσειράΕπιλέξιμες γραμματοσειρές για το κελί
Μέγεθος γραμματοσειράςΜέγεθος σε στιγμές (pt)
Έντονη / Πλάγια / ΥπογραμμισμένηΜέσω των κουμπιών B / I / U
Χρώμα κειμένουΕλεύθερη επιλογή χρώματος
ΣτοίχισηΑριστερά, κέντρο ή δεξιά

Περιγράμματα κελιών

Τα περιγράμματα κάθε κελιού μπορούν να ρυθμιστούν ξεχωριστά.

ΙδιότηταΕπιλογές
ΠλευρέςΠάνω, δεξιά, κάτω, αριστερά (μεμονωμένα ή όλες)
Στυλ γραμμήςΛεπτή, μεσαία, παχιά, διακεκομμένη, διπλή
Χρώμα γραμμήςΕλεύθερη επιλογή

Τα περιγράμματα ρυθμίζονται στον επεξεργαστή στηλών στην ενότητα Περιγράμματα.

Ρυθμίσεις σελίδας

Στην καρτέλα Γενικά μπορούν να ρυθμιστούν οι επιλογές εκτύπωσης:

  • Προσανατολισμός: Κατακόρυφος (Portrait) ή οριζόντιος (Landscape)
  • Μέγεθος χαρτιού: A4, A3, B4 (JIS), B5 (JIS) ή Letter
  • Περιθώρια σελίδας: Πάνω, δεξιά, κάτω, αριστερά σε mm

Επιλογές παρουσίασης (Σχεδιαστής)

Στην καρτέλα Σχεδίαση, όταν η πηγή δεδομένων είναι Πρόγραμμα εργασίας (Schedule), είναι διαθέσιμες πρόσθετες επιλογές:

  • Ενοποίηση εγγραφών ανά ημέρα: Πολλαπλές εγγραφές της ίδιας ημέρας ομαδοποιούνται σε μία γραμμή
  • Εμφάνιση πλήρους περιόδου: Εμφανίζονται όλες οι ημέρες της περιόδου, ακόμα κι αν δεν υπάρχουν εγγραφές

Πηγές δεδομένων

Διαθέσιμες πηγές δεδομένων για αναφορές:

Πηγή δεδομένωνΠεριγραφή
Πρόγραμμα εργασίας (Schedule)Βάρδιες και προγράμματα εργασίας
Διευθύνσεις (Λίστα)Όλες οι διευθύνσεις ως πίνακας
Διεύθυνση (Λεπτομέρεια)Μεμονωμένη διεύθυνση
ΑπουσίεςΕπισκόπηση απουσιών
Βάρδιες (Shifts)Πίνακας βαρδιών
ΟμάδεςΕπισκόπηση ομάδων
Πρότυπο κοντέινερΠρότυπο για αναφορές κοντέινερ

Η πηγή δεδομένων επιλέγεται στην καρτέλα "Πηγή δεδομένων" και καθορίζει τα διαθέσιμα πεδία.
