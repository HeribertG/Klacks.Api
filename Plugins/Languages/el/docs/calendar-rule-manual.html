Μορφή κανόνων

Υπάρχουν δύο τύποι κανόνων: σταθερές ημερομηνίες και ημερομηνίες σχετικές με το Πάσχα.

Σταθερές ημερομηνίες

Μορφή: MM/DD

ΠαράδειγμαΣημασία
01/011 Ιανουαρίου (Πρωτοχρονιά)
12/2525 Δεκεμβρίου (Χριστούγεννα)
08/011 Αυγούστου (Εθνική Εορτή Ελβετίας)

Σταθερές ημερομηνίες με μετατόπιση ημέρας εβδομάδας

Μορφή: MM/DD+XX+WW ή MM/DD+XX-WW

  • +XX = πρόσημο + μετατόπιση σε ημέρες (00 έως 99)
  • + = επόμενη ημέρα εβδομάδας μετά την ημερομηνία
  • - = προηγούμενη ημέρα εβδομάδας πριν την ημερομηνία
  • WW = ημέρα εβδομάδας (SU, MO, TU, WE, TH, FR, SA)
ΠαράδειγμαΣημασία
11/01+00+THΠρώτη Πέμπτη Νοεμβρίου
11/22+00+THΤέταρτη Πέμπτη Νοεμβρίου (Ημέρα Ευχαριστιών)
05/25+00-MOΤελευταία Δευτέρα Μαΐου (Ημέρα Μνήμης)

Μορφή υποκανόνων

Οι υποκανόνες επιτρέπουν την αυτόματη μετατόπιση μιας αργίας όταν πέφτει σε συγκεκριμένη ημέρα εβδομάδας.

Μορφή: WW+X ή WW-X, πολλαπλοί κανόνες χωρίζονται με ;

  • WW = ημέρα εβδομάδας (SU, MO, TU, WE, TH, FR, SA)
  • +X = μετακίνηση X ημέρες μπροστά
  • -X = μετακίνηση X ημέρες πίσω
ΠαράδειγμαΣημασία
SA+2;SU+1Αν Σάββατο: +2 ημέρες (Δευτέρα), αν Κυριακή: +1 ημέρα (Δευτέρα)
SU+1Αν Κυριακή: μετακίνηση σε Δευτέρα

Σημείωση: Οι υποκανόνες χρησιμοποιούνται τυπικά για τη μεταφορά αργιών που πέφτουν σε σαββατοκύριακο στην επόμενη εργάσιμη ημέρα.
