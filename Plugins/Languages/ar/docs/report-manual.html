دليل مصمم التقارير

هيكل التقرير

يتكون التقرير من ثلاثة أقسام:

  • الترويسة – تُطبع في أعلى كل صفحة
  • المتن – يحتوي على جداول البيانات
  • التذييل – يُطبع في أسفل كل صفحة

يمكن تحرير كل قسم عبر علامة التبويب التصميم في محرر التقارير.

قسم الترويسة

تحتوي الترويسة على ثلاث مناطق: يسار ووسط ويمين.

يمكن أن تحتوي كل منطقة على:

  • الحقول – حقول قاعدة البيانات التي تُملأ تلقائياً
  • نص حر – نص يتم إدخاله يدوياً
  • الصور – مثلاً شعار الشركة (يتم تحميله عبر مدير الصور)

يمكن إضافة الصفوف أو حذفها عبر شريط الأدوات.

جداول المتن

يحتوي المتن على جدول واحد أو أكثر.

الإجراءالوصف
إضافة عمودعبر الرمز + أو بالسحب من قائمة الحقول
عرض العموديُضبط بسحب حدود العمود
ترتيب الأعمدةيتغير بالسحب والإفلات
حذف عمودفي محرر الأعمدة عبر رمز الحذف

يرتبط كل جدول بمجموعة بيانات (مثلاً جدول العمل أو العناوين).

دمج الحقول (Merged Fields)

تسمح الحقول المدمجة بربط بيانات متعددة في عمود واحد.

  • في محرر الأعمدة، انقر على إضافة حقل لإضافة حقول إضافية إلى نفس العمود
  • تُعرض القيم الواحدة أسفل الأخرى أو مع فاصل
  • يمكن تكوين الفاصل في مربع حوار الحقل (مثلاً فاصلة أو مسافة أو سطر جديد)

تذييل الجدول

يمكن عرض تذييل أسفل كل جدول.

  • حقول SUM: يمكن جمع الأعمدة الرقمية تلقائياً
  • التسمية: يمكن إظهار أو إخفاء وصف المجموع
  • يتم التفعيل في محرر الأعمدة ضمن علامة التبويب التذييل

صفوف النص الحر

يمكن إدراج صفوف نص حر قبل جداول البيانات أو بعدها.

الخاصيةالوصف

المحتوىنص حر مثل ملاحظات أو شروط أو حقول توقيع
النمطالخط والحجم والمحاذاة قابلة للتكوين

حدود الخلايا

يمكن تكوين حدود كل خلية بشكل فردي.

الخاصيةالخيارات
الجوانبأعلى، يمين، أسفل، يسار (بشكل فردي أو الكل)

لون الخطقابل للاختيار بحرية

يتم تكوين الحدود في محرر الأعمدة ضمن الحدود.

إعدادات الصفحة

في علامة التبويب عام يمكن ضبط خيارات الطباعة:

  • الاتجاه: عمودي (Portrait) أو أفقي (Landscape)
  • حجم الورق: A4 أو A3 أو B4 (JIS) أو B5 (JIS) أو Letter
  • هوامش الصفحة: علوي، يمين، سفلي، يسار بالمليمتر

خيارات العرض (المصمم)

في علامة التبويب التصميم، عندما يكون مصدر البيانات جدول العمل (Schedule)، تتوفر خيارات إضافية:

  • دمج المدخلات حسب اليوم: يتم تجميع عدة مدخلات لليوم نفسه في صف واحد
  • عرض الفترة الكاملة: يتم عرض جميع أيام الفترة حتى لو لم تكن هناك مدخلات
