دليل برمجة الماكرو (Scripting)

مقدمة

يتيح محرر الماكرو كتابة نصوص برمجية تُستخدم كأساس للحساب للخدمات والأنشطة العملية وغيرها.

أمثلة على الاستخدام:

  • حساب الساعات الدقيق للعطل أو الخدمة العسكرية وفقاً لدرجة التوظيف
  • حساب علاوات العمل الليلي وعطلات نهاية الأسبوع وأيام العطل الرسمية
  • التكيف المرن مع المتطلبات الجديدة

ملاحظة: يعمل مفسر النصوص البرمجية في بيئة معزولة آمنة (sandbox) دون الوصول إلى نظام الملفات أو الشبكة. لا يمكن تنفيذ أي تعليمات برمجية خبيثة.

هياكل التحكم

مهم: يجب كتابة الكلمات المفتاحية مثل ENDIF، ENDFUNCTION، ENDSUB متصلة (وليس END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "grande"
ELSEIF x > 5 THEN
    OUTPUT 1, "mediano"
ELSE
    OUTPUT 1, "pequeño"
ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Fin de semana"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Día laborable"
    CASE ELSE
        OUTPUT 1, "Desconocido"

حلقة FOR-NEXT

FOR i = 1 TO 10
    suma += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR

حلقة DO-LOOP

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

الدوال والإجراءات

مهم: يجب تعريف الدوال (FUNCTION) والإجراءات (SUB) قبل استدعائها لأول مرة. يقرأ المفسر الشيفرة من الأعلى إلى الأسفل: لا تكون الدالة أو الإجراء معروفاً إلا بعد معالجة تعريفه.

SUB (إجراء بدون قيمة إرجاع)

SUB calcular(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (دالة بقيمة إرجاع)

FUNCTION duplicar(x)
    duplicar = x * 2
ENDFUNCTION

DIM resultado
resultado = duplicar(21)
OUTPUT 1, resultado

مثال: حساب العلاوات

يحسب علاوات العمل الليلي وأيام العطل الرسمية وعطلات نهاية الأسبوع مع معالجة صحيحة للمناوبات التي تعبر منتصف الليل (بتقسيمها إلى مقاطع).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Calcular horas nocturnas (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Sá = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Do = 7 (ISO-8601)

    ' Tasa más alta para horas nocturnas
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Tasa más alta para horas diurnas
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Calcular el día de la semana del día siguiente (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Turno que cruza la medianoche - 2 segmentos
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Turno diurno normal
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

الشرح

  • التقسيم إلى مقاطع: بالنسبة للمناوبات التي تعبر منتصف الليل (مثلاً 22:00-06:00)، يتم تقسيم المناوبة إلى مقطعين
  • CalcSegment: تحسب علاوة مقطع واحد مع مراعاة علاوات العمل الليلي والعطل ونهاية الأسبوع
  • نسب منفصلة للسبت/الأحد: sarate وsorate تسمحان بعلاوات مختلفة للسبت والأحد
  • أعلى علاوة: تُطبَّق دائماً أعلى علاوة قابلة للتطبيق
  • holidaynextday: يأخذ في الحسبان ما إذا كان اليوم التالي عطلة (مهم للمناوبات الليلية)
