دليل مزود الهوية

مقدمة

تتيح مزودات الهوية مصادقة المستخدمين عبر خدمات الدليل الخارجية مثل LDAP وActive Directory أو OAuth2/OpenID Connect.

أنواع المزودات المدعومة:

  • LDAP - بروتوكول الوصول الخفيف للدليل القياسي
  • Active Directory - Microsoft Active Directory
  • OAuth2 - إطار تفويض OAuth 2.0
  • OpenID Connect - طبقة هوية فوق OAuth 2.0

استيراد العملاء

عند تفعيل "استخدام لاستيراد العملاء"، يتم استيراد البيانات التالية:

  • اسم العائلة - sn (اسم العائلة) أو displayName
  • الاسم الأول - givenName
  • المسمى الوظيفي - title (المسمى الوظيفي)
  • الشركة - company
  • العنوان - street/streetAddress وpostalCode وl (المدينة) وst (الولاية) وc (الدولة)
  • الجنس - من خلال personalTitle (السيد/السيدة) في Active Directory

ملاحظة: في OpenLDAP يكون الجنس افتراضياً "محايد" ويمكن تصحيحه يدوياً.

نصائح

استخدام Synology NAS كخادم LDAP

يمكن استخدام جهاز Synology NAS كخادم LDAP. لهذا الغرض، يجب تثبيت حزمة "LDAP Server" أو حاوية Docker تحتوي على OpenLDAP.

  • نوع المزود: "Active Directory" (للمصادقة الاحتياطية)
  • مرشح المستخدم: (objectClass=inetOrgPerson)
  • صيغة DN للمستخدم: cn=اسم_المستخدم,ou=users,dc=example,dc=com

اختبار الاتصال

استخدم زر "اختبار الاتصال" للتحقق من الإعدادات. إذا نجح الاتصال، يظهر عدد المستخدمين الذين تم العثور عليهم وبعض أسماء المستخدمين كأمثلة.
