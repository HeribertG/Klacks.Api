Håndbok for makroskript

Introduksjon

Makroeditoren gjør det mulig å skrive skript som brukes som beregningsgrunnlag for tjenester, arbeidsaktiviteter osv.

Brukseksempler:

  • Nøyaktig timeberegning for ferie eller militærtjeneste basert på stillingsprosent
  • Beregning av tillegg for nattarbeid, helger og helligdager
  • Fleksibel tilpasning til nye krav

Merk: Skripttolken kjører i et sikkert sandkassemiljø uten tilgang til filsystemet eller nettverket. Det er ikke mulig å kjøre ondsinnet kode.

Alle variabler og konstanter er typeløse.

Kontrollstrukturer

Viktig: Nøkkelord som ENDIF, ENDFUNCTION, ENDSUB må skrives som ett ord (ikke END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "stor"
ELSEIF x > 5 THEN
    OUTPUT 1, "middels"
ELSE
    OUTPUT 1, "liten"
ENDIF

IF på én linje

IF x > 10 THEN OUTPUT 1, "stor" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Helg"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Hverdag"
    CASE ELSE
        OUTPUT 1, "Ukjent"
END SELECT

FOR-NEXT-løkke

FOR i = 1 TO 10
    sum += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

DO-LOOP-løkke

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Funksjoner og prosedyrer

Viktig: Funksjoner (FUNCTION) og prosedyrer (SUB) må defineres før det første kallet. Tolken leser koden fra topp til bunn: en funksjon eller prosedyre er først kjent etter at definisjonen er behandlet.

SUB (prosedyre uten returverdi)

SUB beregn(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (funksjon med returverdi)

FUNCTION doble(x)
    doble = x * 2
ENDFUNCTION

DIM resultat
resultat = doble(21)
OUTPUT 1, resultat

Eksempel: Tilleggsberegning

Beregner tillegg for nattarbeid, helligdager og helger med korrekt håndtering av vakter som krysser midnatt (segmentdeling).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Beregn nattetimer (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' Lø = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Sø = 7 (ISO-8601)

    ' Høyeste sats for nattetimer
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Høyeste sats for dagtimer
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Beregn ukedagen for neste dag (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Vakt som krysser midnatt - 2 segmenter
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Normal dagvakt
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Forklaring

  • Segmentdeling: For vakter som krysser midnatt (f.eks. 22:00-06:00) deles vakten i to segmenter
  • CalcSegment: Beregner tillegget for et segment med hensyn til natt-, helligdags- og helgetillegg
  • Separate satser for lørdag/søndag: sarate og sorate tillater forskjellige tillegg for lørdag og søndag
  • Høyeste tillegg: Det høyeste gjeldende tillegget brukes alltid
  • holidaynextday: Tar hensyn til om neste dag er en helligdag (viktig for nattevakter)
