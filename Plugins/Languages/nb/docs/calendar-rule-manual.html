Regelformat

Det finnes to typer regler: Faste datoer og Påskerelaterte datoer.

Faste datoer

Format: MM/DD

EksempelBetydning
01/011. januar (nyttårsdag)
12/2525. desember (juledag)
08/011. august (sveitsisk nasjonaldag)

Faste datoer med ukedagsforskyving

Format: MM/DD+XX+WW eller MM/DD+XX-WW

  • +XX = fortegn + forskyvning (antall dager) (00 til 99)
  • + = neste ukedag etter datoen
  • - = forrige ukedag før datoen
  • WW = ukedag (SU, MO, TU, WE, TH, FR, SA)
EksempelBetydning
11/01+00+THFørste torsdag i november
11/22+00+THFjerde torsdag i november (Thanksgiving)
05/25+00-MOSiste mandag i mai (Memorial Day)

Påskerelaterte datoer

Format: EASTER+XX eller EASTER-XX

EksempelBetydning
EASTER+00Påskesøndag
EASTER-02Langfredag (2 dager før påske)
EASTER+01Andre påskedag (1 dag etter påske)
EASTER+39Kristi himmelfartsdag (39 dager etter påske)
EASTER+49Pinse (49 dager etter påske)
EASTER+50Andre pinsedag (50 dager etter påske)
EASTER+60Corpus Christi (60 dager etter påske)

Underregelformat

Underregler gjør det mulig å automatisk flytte helligdager når de faller på bestemte ukedager.

Format: WW+X eller WW-X, flere regler skilles med ;

  • WW = ukedag (SU, MO, TU, WE, TH, FR, SA)
  • +X = flytt X dager fremover
  • -X = flytt X dager bakover
EksempelBetydning
SA+2;SU+1Lørdag: +2 dager (mandag), søndag: +1 dag (mandag)
SU+1Søndag: flyttes til mandag

Merk: Underregler brukes vanligvis for å flytte helligdager som faller i helgen til neste virkedag.
