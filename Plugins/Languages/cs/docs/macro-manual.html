Příručka skriptování maker

Úvod

Editor maker umožňuje psát skripty, které slouží jako výpočetní základ pro služby, pracovní činnosti atd.

Příklady použití:

  • Přesný výpočet hodin pro dovolenou nebo vojenskou službu podle úvazku
  • Výpočet příplatků za noční práci, víkendy a svátky
  • Flexibilní přizpůsobení novým požadavkům

Poznámka: Interpret skriptů běží v bezpečném sandboxovém prostředí bez přístupu k souborovému systému nebo síti. Spuštění škodlivého kódu není možné.

Řídicí struktury

Důležité: Klíčová slova jako ENDIF, ENDFUNCTION, ENDSUB musí být psána dohromady (ne END IF).

IF-THEN-ELSE

IF x > 10 THEN
    OUTPUT 1, "velký"
ELSEIF x > 5 THEN
    OUTPUT 1, "střední"
ELSE
    OUTPUT 1, "malý"
ENDIF

IF na jednom řádku

IF x > 10 THEN OUTPUT 1, "velký" ENDIF

SELECT CASE

SELECT CASE weekday
    CASE 6, 7
        OUTPUT 1, "Víkend"
    CASE 1, 2, 3, 4, 5
        OUTPUT 1, "Pracovní den"
    CASE ELSE
        OUTPUT 1, "Neznámý"
END SELECT

Smyčka FOR-NEXT

FOR i = 1 TO 10
    soucet += i
NEXT

FOR i = 10 TO 1 STEP -1
    IF x > y THEN EXIT FOR
NEXT

Smyčka DO-LOOP

DO WHILE a > 0
    a -= 1
LOOP

DO
    x += 1
LOOP UNTIL x >= 10

Funkce a procedury

Důležité: Funkce (FUNCTION) a procedury (SUB) musí být definovány před prvním voláním. Interpret čte kód shora dolů: funkce nebo procedura je známa až poté, co byla její definice zpracována.

SUB (procedura bez návratové hodnoty)

SUB vypocet(a, b)
    IF a = b THEN EXIT SUB ENDIF
    OUTPUT 1, a + b
ENDSUB

FUNCTION (funkce s návratovou hodnotou)

FUNCTION zdvojnasob(x)
    zdvojnasob = x * 2
ENDFUNCTION

DIM vysledek
vysledek = zdvojnasob(21)
OUTPUT 1, vysledek

Příklad: Výpočet příplatků

Vypočítá příplatky za noční práci, svátky a víkendy se správným zpracováním směn přesahujících půlnoc (rozdělení na segmenty).

IMPORT hour, fromhour, untilhour
IMPORT weekday, holiday, holidaynextday
IMPORT nightrate, holidayrate, sarate, sorate

FUNCTION CalcSegment(StartHour, EndHour, IsHoliday, WeekdayNum)
    DIM SegmentHours, NightHours, NonNightHours
    DIM NRate, DRate, HasHoliday, IsSaturday, IsSunday

    SegmentHours = EndHour - StartHour
    IF SegmentHours < 0 THEN SegmentHours = SegmentHours + 24 ENDIF

    ' Výpočet nočních hodin (23:00-06:00)
    NightHours = 0
    IF StartHour < 6 THEN NightHours = NightHours + IIF(EndHour < 6, EndHour, 6) - StartHour ENDIF
    IF EndHour > 23 OrElse EndHour < StartHour THEN NightHours = NightHours + IIF(StartHour > 23, 24 - StartHour, 24 - 23) ENDIF
    NonNightHours = SegmentHours - NightHours

    HasHoliday = IsHoliday
    IsSaturday = WeekdayNum = 6    ' So = 6 (ISO-8601)
    IsSunday = WeekdayNum = 7      ' Ne = 7 (ISO-8601)

    ' Nejvyšší sazba pro noční hodiny
    NRate = nightrate
    IF HasHoliday AndAlso holidayrate > NRate THEN NRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > NRate THEN NRate = sarate ENDIF
    IF IsSunday AndAlso sorate > NRate THEN NRate = sorate ENDIF

    ' Nejvyšší sazba pro denní hodiny
    DRate = 0
    IF HasHoliday AndAlso holidayrate > DRate THEN DRate = holidayrate ENDIF
    IF IsSaturday AndAlso sarate > DRate THEN DRate = sarate ENDIF
    IF IsSunday AndAlso sorate > DRate THEN DRate = sorate ENDIF

    CalcSegment = NightHours * NRate + NonNightHours * DRate
ENDFUNCTION

DIM TotalBonus, WeekdayNextDay

' Výpočet dne v týdnu pro další den (ISO-8601)
WeekdayNextDay = (weekday MOD 7) + 1

IF untilhour <= fromhour THEN
    ' Směna přesahující půlnoc - 2 segmenty
    TotalBonus = CalcSegment(fromhour, 24, holiday, weekday)
    TotalBonus = TotalBonus + CalcSegment(0, untilhour, holidaynextday, WeekdayNextDay)
ELSE
    ' Normální denní směna
    TotalBonus = CalcSegment(fromhour, untilhour, holiday, weekday)
ENDIF

OUTPUT 1, Round(TotalBonus, 2)

Vysvětlení

  • Rozdělení na segmenty: Pro směny přesahující půlnoc (např. 22:00-06:00) se směna rozdělí na dva segmenty
  • CalcSegment: Vypočítá příplatek pro segment s ohledem na noční, sváteční a víkendové příplatky
  • Oddělené sazby pro sobotu/neděli: sarate a sorate umožňují různé příplatky pro sobotu a neděli
  • Nejvyšší příplatek: Vždy se použije nejvyšší platný příplatek
  • holidaynextday: Zohledňuje, zda je následující den svátkem (důležité pro noční směny)
